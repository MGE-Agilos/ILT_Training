Create Visualizations with Qlik Sense

(Optional) Activity 4: Explore your app on tablets and small-screen devices

Summary:

  • Understand how to prepare the app for small screen devices.
  • Explore and simulate user access with different screen sizes.

Module: Sharing insights, monitoring and finalizing the app

Qlik Sense app: ABC Sales Dashboard v2

Challenge

In this activity, you will be given an opportunity to test how your app displays on small-screen devices.

Google Chrome’s Developer Tools enable you to go through popular device screens without leaving the browser environment. All you have to do is select a device from the Console Emulation tab.

You also have full control of the emulation parameters, such as screen resolution, touch emulation, user agent, sensors, and many more.

To perform your test, complete the following steps:

  1. With one of the ABC Sales Dashboard v2 app sheets open, launch the Developer tools on Google Chrome.
  2. Use the Toggle device mode to select different mobile types and review the app.

Hints

  • Open a sheet in analysis mode in order to use the developer tools.

 

Step-by-step guide
  1. Open the Product overview sheet, in the ABC Sales Dashboard v2 app. Make sure you are not in edit mode.

  2. Open the Developer tools and under the Toggle Device Mode or Emulation, select the device model by changing the Responsive option in the drop-down menu.

    Qlik Sense Desktop:

    1. Access the Qlik Sense Hub in a Google Chrome browser with the URL: http://localhost:4848/hub/my/work
    2. Open the app in the Hub.
    3. Follow instructions for all Qlik Sense editions.
    All Qlik Sense editions:
    1. Open the Google Chrome menu at the top-right of your browser window.
    2. Select More tools > Developer tools (or press Ctrl+Shift+I).
    3. Press the Toggle Device Mode button (or press Ctrl+Shift+M).
  3. Notice how the Product overview sheet is now displayed, as if you were using the selected device. In the Emulation pane, select a different model from the Device menu. If necessary, press F5 to reload the page.
  4. Explore the app by opening sheets and interacting with objects.
  5. Close the developer tools to go back to the standard view. Then turn the Edit mode on in the sheet.
  6. In the sheet properties, change the Small screen layout property to Grid view. Then turn the edit mode off.
  7. Open the Developer tools again and explore the differences.
