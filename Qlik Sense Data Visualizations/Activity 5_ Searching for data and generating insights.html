Create Visualizations with Qlik Sense

Activity 5: Searching for data and generating insights

Summary:

  • Gain familiarity with the different searching options in a Qlik Sense app.
  • Make searches in filter panes (fields) and use the Smart search (global search) to find answers to data questions.
  • Learn how to generate insights and search with natural language queries in the app.
  • Use the predefined Analysis Types available on the Insight Advisor.

Module: Overview, selecting and searching data

Qlik Sense app: Vintage movies

Challenge

Using the Vintage Movies app, perform searches and use the Insight Advisor to answer these questions:

  • Question 1: What movies were directed by Francis Ford Coppola in the 1980’s? Search for the arguments one by one — first director, then decade. Try to enter the search terms all at once. Do you get the same results as before?
  • Question 2: What movies were directed by Alfred Hitchcock? Can you find movies directed by Alfred Hitchcock in the 1980’s?
  • Question 3: What movies starred Nicolas Cage or Sofia Coppola? And which of those movies star both (Nicolas Cage and Sofia Coppola)?
  • Question 4: What movies contain “Godfather” in the title or description? From those movies, who directed the ones that do not include the title,“The Complete Epic”, their length is under 3 hours (180 minutes), and John Wayne is listed in the movie's cast?
  • Question 5: How many movies (Titles) are directed by more than one director in the 1980's and 1990's? Create a chart with the number of directors per movie for all decades using the Insight Advisor with a search.
  • Question 6: Which directors worked with the highest number of actors? How is the behavior of the average rating along the years, depending on the length range? For answering these questions, create two charts using the Insight Advisor analysis types.

Hints

  • The Smart search (Global search) can be launched from the Smart search icon on the top left corner or by using the keystroke shortcut Ctrl+F.
  •  You can also perform searches in filter panes or in the fields available in the Selections tool or in the Selection bar.
    • The Search tool allows you to search in values of all fields in the data model, and make selections on the results that match the search criteria.
    • In the Selections tool you can search in fields that are not represented on a sheet.
  • The Insight Advisor (visual search) can also generate visual results for your searches and natural language queries. The Insight Advisor button can be found on the top bar on the left.
    • When using the Insight Advisor, you can use the left panel to choose the master dimensions and measures or fields you want to use in the search criteria, for example: "Title" by "Number of directors".
    • When reviewing results in the Insight Advisor, you can navigate to the sheets where the visualizations were found or add new visualizations if the results did not exist in the app.
    • Advanced searches with natural language can generate results with complex expressions and filters that can be saved in the app as master items for reusable purposes.
    • The Analysis types are a more direct and guided way to create charts, with predefined templates.
Step-by-step guide
  1. In the Vintage Movies app, navigate to the Dashboard sheet and ensure that all selections are cleared.

    2. Question 1

  2. Use the Smart search or Global search (Ctrl+F) and type francis in the search input box. The Smart search results show the Title, Director, and Actor fields. Select Francis Ford in the auto-suggestion list. The search results are filtered down to two possible results: Francis Ford the actor and Francis Ford Coppola the director. Choose the director in the search result.
    3. Note: The search is not case-sensitive. The Smart search displays a set of possible search strings in the auto-suggestion list when you type in the search box.
  3. Search now for 1980's and select the result under Decade in the results list. In the Selections bar, Director and Decade should be selected. Navigate to the Movies Details sheet to see all the movies related with the selection.
  4. Clear all selections and start a new search for francis 1980's. The results of each term are highlighted in a different color. Select the row with Director: Francis Ford Coppola and Decade: 1980’s. The selections are applied.

    5. Question 2

  5. Search for hitch using the Smart search. There are no matches based on the current selections, but select the option Clear selections and search again. Then select Alfred Hitchcock under Director in the search results. Review the titles associated with this selection.
  6. Clear all selections and search now for 1980's hitch. In this case, there are no exact matches but select Show partial matches. From the terms matched list, select Alfred Hitchcock under Director. This provides a list from all decades.

    7. Question 3

  7. Clear all selections and navigate to the Movie selector sheet.
  8. In the Actor filter pane, search for Nicolas + Sofia and select Nicolas Cage and Sofia Coppola. Take a look at the possible values in the Title filter pane. There are five movies listed and you can see the actors for each movie in the final column of the table at the bottom.
  9. Deselect Sofia Coppola to only have Nicolas Cage selected in the Actor filter pane. Then, in the Title filter pane, select all the possible values (all the titles associated with Nicolas Cage).
  10. Clear the selections in the Actor field. The movies are still selected because the selections in the Title field remain. Then, in the Actor filter pane, select Sofia Coppola.
  11. Analyze the selections now in the Title filter pane. It shows the two movies in which both Nicolas Cage and Sofia Coppola appeared. Also notice that one of the three Nicolas Cage movies, The Cotton Club, now appears as selected excluded.

    12. Question 4

  12. Navigate to the Dashboard sheet and clear all selections. Then type godfather in the search tool. There’s only one result matching the search, the Title field. There could be multiple values in the field that match the search criteria. Select the second result to make the selection in the Title field. This result includes the title containing Godfathers as well. You can see that there are 5 movies selected based on the search.
  13. In the Selections bar, click the Title filter and clear the selection for The Godfather: The Complete Epic title. Look at the bar chart in the sheet to confirm the same two directors remain associated with the selected four movies.
  14. Open the Selections tool and locate the Length filter. Then search in this filter, typing <180, and select all the values matching the search criteria. The movie length is displayed in minutes. This search finds all the possible movies with a duration under 3 hours.
  15. In the Actor filter pane, use the search option to find John *. The list shows all actors with John as the first name. Then select John Wayne from the list, and the number of possible titles is reduced to only those where this actor has starred in. Look at the Director filter pane, the only director associated with those movies is John Ford.

    16. Question 5

  16. Clear all selections and select the Insight Advisor. You can type your question directly in the input box at the top or you can select the fields or master items on the left panel.
  17. Type Title by Number in the input box. A suggested option list appears in the search box. From the suggested options, select Number of Directors and press enter to see the results. Complete the search text adding the following: during the decade 1980's and 1990's and press enter.
  18. Review the results. A chart is the displayed as a Matching result. Notice other results are categorized into:
    1. Additional results, new generated charts based on the search. The Add to sheet button at the bottom will add it into a sheet in the app.
    2. Existing charts, links to existing visualizations in the app. The bottom right corner of each result includes the option to navigate to the sheet where the result can be found.
    19. Note: The generated Insights results are new visualizations generated based on your search and the data associations. You can add the new visualizations into your existing app by using the Add to sheet button. From the drop down menu next to this option, you can also choose the sheet the new visualization will be added to.
  19. Select the matching result. It is a bar chart of the number of directors by movie titles for the specified decades. Note that you can perform selections in the chart. Additionally, in the right panel, you can see the dimensions, measures, and filters applied to the chart. Clear the filter to see the data for all decades.
  20. Add the new chart to the Dashboard sheet. Navigate to the sheet using the Go to sheet icon in the bottom right corner of the chart result.
  21. The new chart has been located on this sheet. You can use the Y-axis to select Number of directors from 1.1 to 12 to select all the movies directed by more than one director. Review in the selections bar that there are in total 52 movies directed by more than one director. You can now analyze what movies these are.
  22. Right-click on the bar chart and select View data. The bar chart is transformed into a table. Notice that you can sort the table by the Number of directors by clicking on the column header. Then right-click on the table and select View chart to transform it back to a bar chart.

    23. Question 6

  23. Clear all selections and select the Insight Advisor again. This time, we will select a predefined analysis type from the tiles we have on the start page. Pick Ranking from the Analysis Types. Select Number of Actors from measures, and Director from dimensions.
  24. Add the new chart to a new sheet. Go to that sheet and call it Insights Analysis Types sheet.
  25. Come back to the Insight Advisor and pick the Trend over time analysis. Select Avg rating from measures, Year as Date/time dimensions, and Length Range from dimensions.
  26. Add the new chart to the Insights Analysis Types sheet.
