Create Visualizations with Qlik Sense

Activity 2: Working with set analysis

Summary:

  • Explore and work with the Set analysis syntax in expressions.
  • Use Set analysis in the visualizations in the app.

Module: Controlling data selections

Qlik Sense app: ABC Sales Dashboard v1

Challenge

Create a new sheet, Regional Orders, in the ABC Sales Dashboard v1 app and display the monthly order volume trend for each division.

  • Add the following fields as filters in the sheet: Year, Quarter, Month, Division, and Customer
  • Show the EMEA division monthly order volume using a line chart
  • Show the Americas monthly order volume in another line chart with one line for each division in the Americas region
  • Show value labels and data points in the line charts

Note: Measures and data shown depend on today's date. If you reload your app data, your results might differ from the solution image results.

Hints

  • As the charts you create will be very similar, you can create one and copy it, then make changes to the measures and expressions.
  • The set analysis statement is declared with curly brackets, inside the function.
    • The $ set identifier represents the current selections.
    • The set modifier specifies the field values. For instance: Division={'EMEA'}.
  • Outer set expression can be used instead of inner. In that case, the set analysis statement must be place ahead of the function.

 

Step-by-step guide
  1. In the ABC Sales Dashboard v1 app, create a new sheet called Regional Orders and open it in edit mode.
  2. Add the following fields in a filter pane in the sheet: Year, Quarter, Month, Division, and Customer.

    3. Create the EMEA Line chart

  3. Create a line chart using Month as dimension and Num Orders as measure. Then make sure the Month dimension is set to Sort numerically ascending in the Sorting area and that the Include null values box is unchecked.
  4. Use the properties panel to Show data points and display Value labels, under Appearance > Presentation.

  5. Modify the Num Orders measure by breaking the link to the master item and using the following expression:

    Count({$<Division={'EMEA'}>} distinct OrderID)
    6.  
  6. Add a title to the chart, Order volume trend - EMEA.

    7. Create the Americas Line chart

  7. Copy and paste the existing line chart. You will now have two line charts that are the same.

  8. Select one of the line charts, and change the following:

    • Replace the measure using the following expression:
      Count({$<Division={"*America"}>} distinct OrderID)
    • Change the title, Order volume trend - Americas
    • Add a second dimension: Division

  9. Under Appearance > Colors and legend, clear Show legend title. Then, position legend at the top.

    Question: What if you also want to display the total trend line for Americas? In this case, you could transform the chart as follows:

    Remove the second dimension (Division) from the chart and create two more measures, one for North America and one for South America:

    • Count({$<Division={'North America'}>} distinct OrderID)
    • Count({$<Division={'South America'}>} distinct OrderID)
