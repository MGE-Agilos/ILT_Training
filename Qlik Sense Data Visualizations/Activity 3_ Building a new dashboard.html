Create Visualizations with Qlik Sense

Activity 3: Building a new dashboard

Summary:

  • Explore and work with the Qlik Sense visualizations.
  • Create sheets and sheet objects in a Qlik Sense app to create a well-designed and structure dashboard.
  • Identify the most suitable sheet objects or visualizations to display the business data and metrics.

Module: Creating visual analytics

Qlik Sense app: ABC Sales Analysis v1

Challenge

Review the ABC Dashboard sheet in the ABC Sales Analysis v1 app.

The dashboard gives an immediate overview of the company's performance. Please note, due to the amount of information, this sheet is an extended sheet with a scrollbar on the right.

  • Analyze the sheet:
    • Identify and review the visualizations and the information displayed in the sheet.
    • Make selections, focusing on the EMEA region, to see how the business is doing in this region.
  • Edit the sheet and review the objects and sheet properties.
  • Recreate some of the visualizations in this dashboard in a new sheet called My Dashboard.

Hints

  • In analysis mode, you can analyze the data and make selections using the existing visualizations or the Selection tool.
  • In order to add or modify visualizations in a sheet, the Edit mode must be enabled.
  • Visualizations can be added to the sheet by dragging them from the Assets panel > Charts section.
  • Customize visualizations using the Properties panel. This panel is visible when the visualization is selected and the edit mode is enabled.
  • Master items and Fields can also be dragged from the Assets panel on top of visualizations in order to add or replace dimensions, measures, or color properties in the chart. The Chart suggestion wizard will auto-generate or change visualizations based on the information displayed. You can always turn off the chart suggestions if needed.
  • You can copy and paste objects from different sheets in the app using the Edit bar when the edit mode is enabled.
Note: When editing a sheet, visualizations may not have enough room to display prompts such as Add dimension, Add measure, or Click to add text and measures. If that’s the case, try one of the following:
  • Set the visualization to full-screen mode
  • Increase the size of your Qlik Sense window
  • Close the side panels (Assets and Properties) to get more room
  • Use the Properties panel, instead of the prompts in the visualizations, to add dimensions and measures

 

Step-by-step guide
  1. In the ABC Sales Analysis v1 app, open the ABC Dashboard sheet in analysis mode.

    2. Analyzing the Dashboard sheet

  2. This sheet is divided in three main sections with titles and text:
    • The top part of the sheet displays the main business measures (Sales, Num Orders, COGS, Margin, Margin%, and Discount) for the ABC company using KPIs and Linear Gauges.

      Additionally a summary of the accumulated trend over time for the main measures are displayed using Line charts to represent the sales trends (with full accumulation and normal values) and drill-down dimension (Year > Year Month).

    • The Year on Year Comparison section focuses on Sales over time and performs comparison with YTD, Current YTD, and Last YTD sales measures using:

      • a Combo chart with Sales, Sales YTD, and Margin% by year and month for comparison between these three measures over time.
      • a Bar chart to display the difference between Sales CYTD - LYTD by Product Drill-down (Region > Division > Product Category > Product).
      • a KPI to display the Sales CYTD measure and the Sales CYTD - LYTD.
    • The Sales Composition section displays more detailed information about the sales across some business lines:
      • a Donut (Pie chart) displays the Sales by Product Drill-down and shows the sales composition and sales share between Regions, Divisions, Categories, and Products. The segments are color coded based on the sales measure and uses a maximum of 7 segments for clarity including the rest in "Others" segment.
      • Text and image objects are used to give extra information about the number of products, product categories, as well as the number of customers and cities.
      • the Bar charts are used to show the top 10 products and customers based on sales using this measure to color the bars with a Sequential gradient. Both charts display as well the average with a dotted line.
  3. Select EMEA in the donut chart and inspect how the drill-down dimension works and compare with the solution image. Then select the latest year in the combo chart to see the monthly trend for that year.
  4. Clear all selections in the Selections bar on the top.

    5. Reviewing object and sheet properties

  5. Use the edit mode to modify the sheet by selecting the @ button.
  6. Select the ABC Dashboard sheet title and then review the properties panel. Note how the Grid spacing is set to Narrow and the Extend sheet property is set to On.
  7. The style of all the sheets in the app can be changed from the app properties. Select the icon next to the app name at the top, ABC Sales Analysis, and use the app options button to access the sheet styling for all the sheets in the app. Then chose the Sense classic theme under appearance and use the X button to close the app properties. Then select the app name again to hide this properties panel.
  8. Select other objects in the sheet and then review their properties using the properties panel.

    9. Creating a new dashboard

  9. Create a new sheet and rename it to My Dashboard. Note you can copy and paste sheets between apps or in the same app.
  10. Follow the Reproduce Dashboard sheet steps to reproduce different parts of the dashboard sheet.

Note: You may need to use the Advanced options toggle to access the Assets panel in Edit mode.

    Reproduce Dashboard sheet
  1. In the sheet properties panel, make sure the Grid spacing is set to Narrow. Then turn on the Extend sheet property.

    2. KPIs and high-level trends

  2. From the Assets panel Master items > Measures menu, drag and drop the following measures in the sheet: Sales, Num Orders, COGS, and Margin. Resize them to 6 grids wide x 3 grids high, leaving space at the top right of the sheet.
  3. Repeat the previous step with the Margin% measure, dragging this measure into the sheet twice. The chart suggestions wizard represents this measure as a gauge. Select the KPI representation for the first one using the Chart suggestion menu. Now you have the Margin% measure represented as KPI and gauge objects.
  4. Select the gauge and size it to 6 grids wide and 3 grids high. Note that the configuration options in the properties panel is very limited for this object as the chart suggestions wizard is turned on. Turn off this property using the Chart suggestions toggle to review the full list of properties.
    5. Note: Select a chart to configure the properties using the Properties panel. If you cannot see this panel, click on the h properties button from the top bar to display it.
  5. With the Margin% gauge selected, modify the properties panel:
    1. Under Appearance > Presentation, enter 0.65 as Max and select Bar.
    2. Under Appearance > Measure axis, for Labels and title select None.
  6. Add the Discount measures into the sheet and display it as KPI, then locate it under the Margin% KPI and size it to have the same dimensions as the KPI above. Then, copy the Margin% gauge, using the edit bar or right-click on the visualizations and then select copy. Paste it on to the sheet. A second gauge appears. Move it below the original one.
  7. From the Assets panel, drag the Discount master measure onto the second gauge and select Replace Margin%.
  8. From the Assets panel, under Charts, drag a Line chart and select Year Drill-down as dimension and Sales as Measure. Then, using the line chart properties, modify the following sections:
    1. Under Data > Measures, right click on the Sales measure and select Duplicate. Then, expand the duplicated measure and select Modifier. Then choose Accumulation in the drop down and set the Range to Full.
    2. Under Appearance > General, turn off the Show titles property.
    3. Under Appearance > Presentation > Styling > Chart, set the Line thickness using the slider to increase the thickness of the line and set the Line type to Dashed. Then, go back to the properties menu by selecting the back arrow on the top.

      4. Qlik Sense Client-Managed: If you are working with the Skytap environment, you will see the Line thickness option after clicking on Styling.

    4. Under Appearance > Presentation, select Area, set Scrollbar to None, Show data points, and set the Grid line spacing to Custom (No Lines).
    5. Under Appearance > Y-axis, for labels and title select None.
  9. Adjust the size of the line chart with the same dimensions to match the size of the Sales KPI and locate it underneath the KPI. Then copy and paste the line chart three more times in the sheet and replace the Sales measures for the following measures respectively: Num Orders, COGS, and Margin. You can use the Assets pane, select Master items > Measures and drag the corresponding measure on top of each chart selecting Replace "Sales". Apply the accumulation as modifier in the expression when needed and locate the sheet objects as follows:

    10. Note: Measures and data shown depend on today's date. If you reload your app data, your results might differ from the solution image results.

    Year on Year Comparison section

  10. From the Assets panel > Charts, drag the Text & image visualization below the line charts (in the free space). Enter text and format it as follows:

    Year on Year Comparison

    How much improvement have we made this year versus last year?

  11. From the Assets panel, drag the master measure Sales on a free space in the sheet. A new KPI is created. Then drag the Year Drill-down dimension on top, to convert the visualization into a bar chart.
  12. Drag two new measures on top: Sales YTD and Margin%. The visualization is now a scatter plot.
  13. Drag a Combo chart from the Assets panel on top of the visualization and select Convert to: Combo chart.
  14. Under the Combo chart properties panel, navigate to Appearance > General, turn on Show titles and set the Title to Sales and YTD Sales trend and the Subtitle to Includes Margin% as points and Num orders in tooltip.
  15. Then under Appearance > X-axis, set Continuous to Custom and clear Use continuous scale.
  16. Modify other properties:
    1. Under Data > Dimensions, select the dimension and clear Include null values.
    2. Under Data > Measures, select the Margin% and select Marker and Secondary axis. Make sure the other two measures are in the Primary axis.
    3. Under Sorting, make sure the dimension is the first item in the list.
    4. Under Appearance > Presentation, set Bar labels to Auto.
    5. Under Appearance > Colors and legend, set Legend position to Top.
    6. Under Appearance, change the X-axis to show Labels only.
    7. Under Tooltip, set the toggle to Custom and add the Num orders as extra measure for the tooltip.
  17. Drag the Sales CYTD - LYTD measure on a free space on the sheet, then drag Product Drill-down dimension on top. A new bar chart is created.
  18. Turn off the Chart suggestions in the new bar chart to access the full set of properties and perform the following changes:
    1. Under Sorting, make sure Sales CYTD - LYTD is set first in the list, and then set the sorting in this measure to Custom and Sort numerically Ascending.
    2. Under Appearance > General, add a title (Sales CYTD - LYTD) and a footnote (* Drill-down Division, Category, and Product).
    3. Under Appearance > Presentation, change the bars to Horizontal, and turn on Value labels. Then set the styling of the bars with a small outline width in black color.
    4. Under Appearance > Colors and legend, change Color to Custom, then select By measure, turn off the Library colors, and select Diverging gradient with Reverse colors. Note that you can define the range of values used to color by measure.
    5. Under Appearance > Y-axis show labels only on both the Y-axes and under Appearance > X-axis hide all labels and title on the X-axis.
  19. Drag the Sales CYTD measure on a free space on the sheet. Then drag Sales CYTD - LYTD on top. The new KPI now has two measures. Resize and locate the objects as displayed in the image:

  20. Turn the Chart suggestions off and use the properties panel (Appearance > Color > Second) of the KPI to change the second measure colors:
    1. Turn Conditional colors on and turn Library colors off.
    2. Click Add limit, and set a limit of 0. The segment is now divided into two.
    3. Select the first part of the segment and set a color and a shape. Then repeat the process for the second part of the segment.

    21. Sales Composition section

  21. Click Extend sheet to make more space at the bottom of the dashboard.
  22. Copy and paste the Text & image object, locate the new text under the Combo chart, and change the text as follows:

    Sales Composition

    Where are the Sales coming from?

  23. Include two more Text & image objects to display images and dynamic text using expressions as follows:
    1. In the first Text & image, add the following text: <Exp1> products in <Exp2> categories, where <Exp1> and <Exp2> are:

      Exp1:

      count(distinct ProductID)

      Exp2:

      count(distinct CategoryName)

      Add the products.png image from the ABC images folder as Background image for this first Text & image object.

    2. In the second Text & image, add the following text:<Exp1> customers in <Exp2> cities, where <Exp1> and <Exp2> are:

      Exp1:

      count(distinct CustomerID)

      Exp2:

      count(distinct City)

      Add the customers.png image from the ABC images folder as Background image for this second Text & image object.

    3. Resize and distribute the objects as shown in the image below:

    Note: Images can be added or used in the Qlik app using the Media Library F. In the app, these images have been already added.

  24. Include two more visualizations:
    1. Pie chart using Product Drill-down as dimension and Sales as measure. Use the properties to:
      1. Limit the dimension to the Fixed value of the Top 7, with Show others selected to display the Others segment.
      2. Sort by Sales numerically descending.
      3. Represent it as Donut chart and apply different styling (use the Styling button to set different inner and corner radius and change the outline color).
      4. Set the Color to Custom by measure (Sales) with a Sequential gradient (by turning off the Library colors).
      5. Clear Show legend.
      6. Add a title, Sales share.
    2. Bar chart using Product as dimension and Sales as measure. Use the properties to:
      1. Under Data > Measures, expand the Sales measure and click Add trend line, then select the Average.
      2. Limit the dimension to the Top 10 values with no Others segment.
      3. Sort by Sales numerically descending
      4. Set Value labels to Auto.
      5. Set the bar orientation to horizontal and the Color to Custom by measure (Sales) with Sequential gradient.
      6. Clear Show legend and set the X-axis to None (labels and title).
      3. Note: Value labels might not be displayed in the visualization when the object size is not big enough. You can expand the object to full screen to see the value labels.
  25. Copy and paste the last bar chart and, in the copy, change the dimension to Customer. Change the title and make sure you limit the number of customers to 10 and clear the Others bar.
