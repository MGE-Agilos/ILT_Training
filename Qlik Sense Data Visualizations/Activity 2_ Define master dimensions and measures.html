Create Visualizations with Qlik Sense

Activity 2: Define master dimensions and measures

Summary:

  • Define new dimensions and measures in apps creating them as master items.
  • Understand master dimensions and drill-down dimensions
  • Familiarize with the expression syntax and the drill down dimensions.

Module: Preparing data for analysis

Qlik Sense app: ABC Sales

Dependencies: Before working in this activity, make sure you complete Activity 1.

Challenge

In the ABC Sales app, define reusable master items by creating the following master dimensions and measures.

Master dimensions

Use the specified names for the master dimensions:

Table Field Master dimension name
products CategoryName Product category
customers Country Country
customers City City
customers Customer Customer
customers

Division

Division
employees EmpName Sales person
orders OrderDate.autoCalendar.Year* Year
orders OrderDate.autoCalendar.Quarter* Quarter
orders OrderDate.autoCalendar.Month* Month
orders OrderDate.autoCalendar.YearMonth* Year Month
products Product Product

Create two additional master dimensions using the Drill-Down capability:

  • Product Drill-down: Region > Division > CategoryName > Product
  • Year Drill-down: OrderDate.autoCalendar.Year* > OrderDate.autoCalendar.YearMonth*

*If the OrderDate.autoCalendar.Field is not available, use the OrderDate.Field.

Master measures

Use the following measure names, calculations, and color codes (when specified) for the master measures:

Master measure name Expression Color

Sales

Sum(Sales)

COGS

Sum(UnitCost * Quantity)

Discount

Num(Avg(Discount),'0.00%')
Margin Sum(Sales - (UnitCost * Quantity))

Margin%

Num(Sum(Sales - (UnitCost * Quantity)) / Sum(Sales),'0.00%')
Num Orders Count(Distinct OrderID)

Note the Num() function is used in some expressions to format the numbers. You can also format some of these expressions using the Number formatting option.

Drag the new measures into the sheet to check the expressions are evaluating correctly.

Hints

  • Master items can be added to the app in different ways:
    • From the Assets panel (visible when editing a sheet), under the Master items section
    • From the Assets panel (visible when editing a sheet), under the Fields section, by right-clicking on the field
    • From the Data model viewer, under the Preview panel, when the field is selected
    • From the Properties panel of an object, when you have selected some field to be used as a measure or dimension, and you want to add it as a master item. Under the Data section, when the measure or the dimension is selected, by clicking on Add new.
  • A hierarchy of fields can be defined as a drill-down master dimension. Drill-down dimensions are created as master dimensions using the Drill-down option radio button of the Create new dimensions dialog.
  • When editing the sheet, you can drag and drop fields and master dimensions and measures into the sheet canvas. The sheet can be analyzed by toggling out of edit mode.
Step-by-step guide
  1. In the ABC Sales app, navigate to the Sheet. The My new sheet opens as an empty sheet, because we haven't created any sheets in the app. Click Edit sheet to enable the edit mode and edit the sheet.

    2. Note: You may need to use the Advanced options toggle to access the Assets panel in Edit mode.

    Defining master dimensions

  2. From the Assets panel, under éMaster items, select Dimensions and then click Create new.
  3. Select one of the fields in the first table above (you can search for the field), and then click Create.
  4. Repeat steps 2 and 3 for each dimension in the Master dimensions table, changing the dimension name as required.
  5. From the Assets panel, under Master items > Dimensions, create a new master dimension.
  6. In the Create new dimensions dialog, select Drill-down and add Region, Division, CategoryName, and Product in that order to insert them into the Field list. Name the dimension as Product drill-down. Then click Create.
    7. Note: You can rearrange the order of the fields you have selected, if necessary, using drag-and-drop in the list of selected fields.
  7. Repeat steps 5 and 6 for the Year drill-down dimension. Make sure you add the OrderDate.autoCalendar.Year and OrderDate.autoCalendar.YearMonth in that order.
    8. Note: The OrderDate.Year and OrderDate.YearMonth fields appear as OrderDate.autoCalendar.Year and OrderDate.autoCalendar.YearMonth under the Field section of the dialog box.

    Defining master measures

  8. From the Assets panel, under é Master items, select Measures, then click Create new.
  9. Type the corresponding Expression and Name for the measure (based on the Master measures table above). You can copy/paste the expression instead of typing it, if needed.
  10. If the measure has a specified color, choose the color from the Measure color picker.
  11. Repeat steps 9 and 10 for each measure in the table. Notice that master measures can be duplicated by right-clicking on them. This can be useful when creating similar measures, like Margin and Margin%, as you can edit the copied measure.

    12. Reviewing the data

  12. Drag and drop the new measures into the sheet and compare the results with the solution image. Notice how the assigned color is displayed for each measure. Make sure you represent all as KPIs. You can use the Chart suggestions on the properties panel, by clicking on Auto, and then you will see the Recommended charts, where you will be able to select the chart type you prefer.
  13. With the Shift key pressed, drag and drop some of the dimensions into the sheet to use them as filters.
    Note: Add fields or dimensions as filter panes into the sheet by holding the shift key when you drag and drop the items into the sheet. This prevents Chart suggestions. Master expressions defined in the app can be used in chart expressions. You can copy and paste measures between visualizations.
  14. Then click Edit sheet to enable the analysis mode in the sheet to interact and make selections.
    15. Qlik Sense Client-Managed: Click Done editing to come back to the analysis mode.

Solution

Click on the Solution image to open as a pop-up, or right-click on the image for more options. To close, click on the image again.
