Create Visualizations with Qlik Sense

Activity 3: Working with alternate states

Summary:

  • Explore Qlik Sense alternate states in the app.
  • Use alternate states in different objects and sheets.
  • Create new expressions using alternate states.

Module: Controlling data selections

Qlik Sense app: ABC Sales Dashboard v1

Dependencies: Before working in this activity, make sure you complete Activity 2.

Challenge

In the ABC Sales Dashboard v1 app, create a chart comparison in the Regional orders sheet:

  • Create a monthly order volume trend line chart to compare two sets based on user selections
  • Use set analysis in the charts (in the measure and in the reference line expression):

    • Use a set identifier for the current selections.

    • Use a modifier that selects the value in the Division field.
    • Use alternate states combined with set analysis for the comparison trend.
  • Show value labels and data points in the line charts

Bonus point:

Create a new sheet called Set comparison. Business users will perform sales and margin analysis comparing these metrics between different set of products they select:

  • Use two alternate states and two filter panes in the sheet to allow users to select products.
  • Display two KPIs with the Sales and Margin% for each alternate state.
  • Visualize in bar charts the sales measure (using the YearMonth Drill-down and the Division dimensions) comparing the two alternate sets.

Note: Measures and data shown depend on today's date. If you reload your app data, your results might differ from the solution image results.

Hints

  • In the chart, use set expressions in each measure to assign different selection states to each calculation.
  • The set analysis statement is declared with curly brackets, inside the function.
    • The $ set identifier represents the current selections.
    • Alternate state names can be used as set identifiers in your set analysis expression.
    • The set modifier specifies the field values. For instance: Division={'EMEA'}.
  • Outer set expression can be used instead of inner. In that case, the set analysis statement must be place ahead of the function.
  • Alternate states can be defined in the app when editing a sheet, using the Assets panel > Master items section.
  • Assign an alternate state to a sheet object using the object properties panel (under the Alternate states section).
Step-by-step guide

    Create a comparison chart

  1. Drag a filter pane into the Regional orders sheet, right below the "Order volume trend - EMEA" line chart. Then add Year and Division as fields.
  2. From the Assets panel > Master items > Alternate states section, use the Create new button to create two new alternate state in the app called Set1 and Set2.
  3. In the properties panel of this filter pane, under Appearance > General, set the Title to "Group1" and under the Alternate states section, set the State to Set1.
  4. Copy and paste the filter pane, and in the copy, change the Title to "Group2" and set the State to Set2.
  5. Copy and paste the EMEA line chart. You will now have 3 line charts. Then change the title to Order volume trend - Comparison.
  6. Remove the existing measure and create two measures as follows:

    First measure: set the Label to Group1 and the expression as follows:

    Count({Set1} DISTINCT OrderID)

    Second measure: set the Label to Group2 and the expression as follows:

    Count({Set2} DISTINCT OrderID)
  7. In analysis mode, select the following in the group filters:
    • Group1 filter pane: select 2023 and the EMEA division
    • Group2 filter pane: select 2023 and the South America and North America divisions

    Question: Would it be possible to have dynamic labels in the measures to see what each line represents in the Order volume trend line chart legend?

    Yes, you can use dynamic labels in the filter panes or in the measures of the line chart. Discuss the differences:

    In the filter pane subtitles:

    =GetFieldSelections( [OrderDate.autoCalendar.Year])&'-'&GetFieldSelections( [Division])

    In the measure labels of each chart:

    =Concat(distinct {Set1} [OrderDate.autoCalendar.Year],',')&'-'& Concat(distinct {Set1} Division,',')
    =Concat(distinct {Setp2} [OrderDate.autoCalendar.Year],',')&'-'& Concat(distinct {Set2} Division,',')

    8. Bonus point:

  8. Create a new sheet in the ABC Sales Dashboard v1 app, and name it: Set comparison.
  9. In edit mode, access the sheet properties panel and, under Alternate states section, make sure the State is set to <default state>.
  10. From the Assets panel > Charts section, drag a Text & Image object into the sheet and add the following text under the sheet title: Select in both category and product filters to analyze two sets side by side.
  11. From the Assets panel > Charts section, drag a Filter pane in the sheet. Use the Product and Product Category fields in the filter pane. Then, in the filter pane's properties panel, under Appearance > Alternate states, set the State to Set1.
  12. Make sure you display the title in the filter pane and write Set 1 as the title.
  13. Copy and paste the filter pane in the same sheet and set this new filter pane's State to Set2 and change the title to Set 2.
  14. From the Assets panel > Master items > Measures section, drag the Sales measure into a free space in the sheet. This measure is represented as a KPI. Then drag also the Sales CYTD - LYTD on top to include a secondary measure in the KPI.
  15. In the KPI properties panel, set the alternate state to Set1. Then copy the KPI and paste it again in the sheet and change the state to Set2 for the new KPI.
  16. From the Assets panel > Charts section, drag a new KPI in the sheet using the Margin% measure and set the alternate state to Set1 in the properties panel. Then copy the Margin% KPI and paste it again in the sheet but change the state to Set2.
  17. Relocate the KPI and filter pane objects in the sheets, so all the objects with the Set1 state are displayed on the left and the objects with the Set2 state are displayed on the right.
  18. Create a new bar chart in the sheet using Year Drill-down as dimension and the following two expressions:
    Sum({Set1}Sales)
    Sum({Set2}Sales)

    19. By adding the set expressions {Set1} and {Set2}, the calculation is performed only over the data with the corresponding alternate state selections applied.

  19. Rename the new expressions to Sales Set1 and Sales Set2 and show the legend at the top.
  20. Copy and paste the chart to duplicate it. In the new bar chart, remove the existing dimension, then include Division as the new dimension and hide legend and Y-axis.

    21. Analyzing the sheet

  21. In analysis mode, make selections in Sportwear category using the filter pane in Alternate state Set1, then in the other filter pane, select all Men's product categories and compare both charts. Compare with the solution image.
