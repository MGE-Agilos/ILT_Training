Create Visualizations with Qlik Sense

Activity 2: Working with stories

Summary:

  • Explore and work with stories in a Qlik Sense app.
  • Create stories and include snapshots and other objects in slides.
  • Understand the navigation and export options in stories.

Module: Sharing insights, monitoring and finalizing the app

Qlik Sense app: ABC Sales Dashboard v2

Dependencies: Before working in this activity, make sure you complete Activity 1.

Challenge

Create a story in the ABC Sales Dashboard v2 app called EMEA Report using the snapshots you captured in the previous activity.

The story should focus on the sales analysis for the EMEA region and should include the following items:

  • Three slides (use an embedded sheet in one of them)
  • At least one Text title and one Shape
  • Snapshots and Effects (use these three effects: Lowest value, Highest value, and Any value)

Play the story and delete one of the slides.

Examine export options.

Bonus point:

  • Add a thumbnail to the story using one of the images in the Default library.

Hints

  • Stories can be created and edited from the Storytelling menu in an app.
  • Snapshots, Text, Shapes, and Images can be added to a slide when editing the story.
  • Effects can be applied on existing snapshots in the slide by dragging them from the Effects library onto the snapshot. For the Any value effect, you can select the specific value you want to highlight.
  • A thumbnail image and a description can be added to the story when editing the info panel of the story.
  • When editing a story, you can use the Global Menu to explore the export options.
Step-by-step guide
  1. In the ABC Sales Dashboard v2 app, create a new story called EMEA Report using the Storytelling option from the toolbar.

    2. First slide

  2. Open the story and drag a Title onto the slide from the Text objects section. Then double-click in it to add the text, Let’s talk about EMEA.
  3. Using the Snapshot library, drag the two KPI snapshots into the slide. Then drag a Paragraph from the Text objects section and add some text to explain the numbers.
  4. Using the Shapes library, drag a World shape next to the KPI for all regions. Set the color to dark blue.
  5. Add the bar chart snapshot onto the slide. The bar chart is displayed in the slide as an image.
  6. Resize the image, using the unlock padlock for free resizing, then click @ edit to hide the legend and click Done.
  7. Using the Shapes library, drag a Flag shape on top of the bar chart pointing at the top bar in red. Resize the flag and select a color for it. (For instance, use a red color)
  8. Drag another Shape and Paragraph next to the bar chart image to put some context in the image: "In the EMEA region, Sportswear category has suffered the hisghest decrease in sales versus last year to date. This might be due to a US sport clothing competitor expanding to the EMEA region".
  9. Highlight the name of the product category. Format part of the text to different color or font if necessary.
  10. Right-click on the bar chart image and select Go to Source. This allows the user to review more details from the story in the app.
  11. Navigate back to the story. The first slide is completed and it will be similar to the example below:

    Slide 1 - Example

    12. Second slide - Duplicate a slide in a story

  12. Duplicate the current slide to create a second slide.
  13. On the second slide, change the title to read the following: More about EMEA.
  14. Delete all the other images, paragraphs, and shapes in the second slide.
  15. Drag a snapshot containing the pie chart with the Sales share - EMEA 2023 annotation twice onto the second slide. Now you have the same snapshot repeated twice on the second slide.
  16. Using the Effects library, drag the Highest value on the first pie chart snapshot. The visualization changes, highlighting the product category within EMEA 2023 with the highest sales share, Sportswear.
  17. Using the Effects library, drag the Lowest value onto the second pie chart snapshot. The visualization changes, highlighting the category within EMEA 2023 with the lowest sales share. Please note if you have "Others" in the chart, go to the source and click on others to see what categories are the lowest. In this case, Men's Footwear and Swimwear. Using the Effects library, drag the Any value, click the effects icon in the snapshot, and select Others from the drop-down.
  18. Drag the combo chart snapshot to the bottom right of the slide and edit the snapshot to remove titles. Then use the Shapes library to add a red circle over the year 2023 in the combo chart.
  19. Add a paragraph to explain the selections in the chart and provide some details. Include also a link to the EMEA bookmark.
    Note: You can include links to URLs or bookmarks within the text in the paragraph. Select the text you want to link and use the é Create link icon. Then you can enter a URL or select an existing Bookmark. For example, choose EMEA.

    20. Slide 2 - Example

    Third slide - Embedded sheet

  20. In the Analyze > Sheet section, navigate to the Product overview sheet and select Baby Clothes and Swimwear as Product categories and the EMEA division, then navigate back to the story (using the Storytelling section) and add the Product overview sheet from the Sheet Library. A third slide is created with the Product overview sheet embedded.

  21. Select the first slide in the story and click Play story. Then use the Actions button at the bottom of the screen to advance to the next sheet or go back to the previous sheet. Notice how you can make selections in the third slide using the filters from the Product overview sheet or charts and hover over the bars to see the tool tips.
  22. Close the Play view to return to editing the story, right-click on the second sheet, and then Delete it.
  23. From the global menu, examine the export options.

    24. Bonus point:

    Add a thumbnail:

  24. From the Stories menu, hover over the EMEA Report story and open the ] Details in edit mode.
  25. Click the image to Change thumbnail and insert an image from the default library.

    26. Note: Stories can be exported to PDF format. Use the Global Menu on the top toolbar to see the export and download options and other actions. Another format available in Qlik Sense on Windows editions is PowerPoint.
