Create Visualizations with Qlik Sense

(Optional) Activity 5: Combining set analysis and bookmarks

Summary:

  • Explore and work with bookmarks and set analysis.
  • Use bookmarks to simplify set analysis syntax.

Module: Controlling data selections

Qlik Sense app: ABC Sales Dashboard v1

Challenge

In the Dashboard sheet of the ABC Sales Dashboard v1 app, create a new visualization displaying the monthly sales trend comparing EMEA vs Americas (includes South America and North America).

Create two bookmarks and then, create two measures in the chart using the them in set analysis (one for the EMEA division and one for the Americas divisions).

Bonus point:

  • Make sure the line chart you create is also affected by the user selections. Use a set operator in set analysis to achieve this.

Hints

  • Line charts are good for representing trends over time. Use a line chart with Month as a dimension.
  • Bookmark names can be used as set identifiers in set analysis, for instance, Sum({Americas} Sales).
  • Outer set expression can be used instead of inner. In that case, the set analysis statement must be place ahead of the function.

 

Step-by-step guide
  1. In the ABC Sales Dashboard v1 app, open the Dashboard sheet in edit mode.
  2. Click Extend sheet to leave some free space at the bottom of the sheet and add a Text & image object with the following text:

    Sales trend

    How are the sales trends going in the business regions?

  3. Click Done editing to enable the analysis mode. Now you can perform selections and searches.
  4. Search for EMEA, and select the value in the Division field. Then click the > Bookmarks button and select Create new bookmark.
  5. Add a title, EMEA, and click Create. Click the > Bookmarks button again to close the panel.
  6. In the selection bar, select the Division field, clear EMEA and select North America and South America values.
  7. Click the > Bookmarks button and select Create new bookmark. Then add a title, Americas, and click Create.
  8. Click the > Bookmarks button again to close the panel.
  9. Add a Line chart below the Text&Image object, in a free space on the sheet. Then select Month as a dimension in the new line chart.

  10. Using the properties panel of the new line chart, add two measures, with the following expressions:

    • Americas:
      Sum({Americas} Sales)
    • EMEA:
      Sum({EMEA} Sales)
  11. Under the Sorting section, make sure Month appears as the first item and it is sorted numerically and ascending.
  12. Under the Appearance > Presentation section, use the Area style, check Show data points, and set Value labels to Auto.
  13. Add a title to the chart, Sales trend Americas vs EMEA.
  14. In analysis mode, use the smart search to select the Q2 quarter. Notice how the line chart does not change with the selection. Discuss how the line chart can change based on the user selections.

    15. Bonus point:

    Use a set operator in the set analysis expressions:

  15. In edit mode, modify the Americas measure in the line chart so the expression is as follows:
    Sum({Americas * $} Sales)
  16. Repeat the previous step for the EMEA measure, so the expression for this measure is as follows:
    Sum({EMEA * $} Sales)

Solution

Click on the Solution image to open as a pop-up, or right-click on the image for more options. To close, click on the image again.
