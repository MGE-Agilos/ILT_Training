Create Visualizations with Qlik Sense

Activity 5: Style your app

Summary:

  • Understand how to review and prepare the app for handing over to consumers.
  • Work with master items, descriptions, and tags.

Module: Sharing insights, monitoring and finalizing the app

Qlik Sense app: ABC Sales Dashboard v2

Challenge

In this activity, you set a thumbnail image for the app and apply a style to all the sheets.

For the thumbnail image, choose any image you like from the media library or from the ABC images folder.

When styling the app, make sure you:

  • Review the available themes. You can also include other existing themes or build your own theme.
  • Apply a background color for the sheet titles using a color range.
  • Change the font color in the sheet titles.
  • Add a logo or an image at the right of the sheet titles.

Feel free to chose your own colors and images. You can also add new images to the library if needed.

Bonus point:

  • Apply a custom theme (available under the Student Files > Themes)
  • Take a screenshot of each sheet in the app in analysis mode and use it as a thumbnail for each sheet

Hints

  • A thumbnail can be added to the app using the app Edit @ panel.
  • In the App options V panel, you can select a standard or custom theme, apply title background color, and add a logo to all sheets. The custom themes need to be added to Qlik Sense before you can apply them.

    • Cloud editions of Qlik Sense: In the management console (https://<your tenant address>/console), under the Themes section, you can add new themes.

    Qlik Sense Enterprise on Windows: Use the QMC > Extensions section to add any new custom themes.

    Qlik Sense Desktop: Create a folder called Themes under: ..\Users\<UserName>\My Documents\Qlik\Sense\Extensions. Then for each theme, make sure you include the theme files (JSON and the QEXT files).

  • New images can also be included in the media library.

    • Qlik Sense Desktop: Copy images manually into the location where the product was installed. By default, the location is under the users' Documents\Qlik\Sense\Content\Default folder.

 

Step-by-step guide
  1. In the App Overview, in the ABC Sales Dashboard v2 app, click on the @ icon to access the app edit panel. In there you can change the name of the app or add a description.
  2. Click on the thumbnail image. The media library dialog appears.
  3. Choose the ABC Sales Dashboard.png image from the In app section. Then click Insert. Note that you can preview the images when you select the image names.

    4. Note: If the image is not in the In app list, you can import it as a new image in the app. You can add new images to the Media library using the Upload media option.

  4. Confirm the changes and open the ABC Dashboard sheet.
  5. Show the app information by clicking on the app name at the top. The app information and the thumbnail are now displayed on the top.
  6. Click the App options V button. Then under Appearance, review the different options for App theme.
  7. Under the App capabilities, make sure both Insight Advisor in hub and Insight Advisor in this app options are turned on.
  8. Under the Sheet title styling, select the left and right colors for the background and a font color compatible with the background color.
  9. Click the 0 icon to select a logo and choose an image from the media library. You can use the ABC company logo.
  10. Make sure the Image alignment is set to right.

    11. Note: Under Appearance and Sheet title styling respectively, you can turn off Show toolbar, as well as Show sheet header. If you want other users to see more clearly and with fewer options the sheets of your app when they access it, they are some options.

  11. Close the app options and navigate through the sheets in the app to see the results.

    12. Bonus point:

  12. Include the custom themes provided in the Student files, and review how the app looks when applying them using the App options > Appearance > App theme. If you have not included these custom themes before, make sure you follow these steps first:
    1. If using Qlik Sense Business or other Saas editions, access the Management console (https://<your tenant address>/console) and add the new themes in the Themes section.
    2. If using Qlik Sense Enterprise on Windows, open the Qlik Management Console, and under Extensions, import each zip file corresponding with each theme.
    3. If using Qlik Sense Desktop, navigate to ..\Users\<UserName>\My Documents\Qlik\Sense\Extensions, and create a folder called Themes. Then unzip all the custom themes in this folder.
  13. In analysis mode, navigate through the sheets taking a screenshot of each sheet and save it in a location you choose. If you are working with Qlik Sense Desktop, use the default folder for images (C:\Users\<user>\Documents\Qlik\Sense\Content\Default).
  14. Open the ABC Dashboard sheet in edit mode and click on the sheet name. Then in the sheet properties panel, click the 0 icon to select a Thumbnail.
  15. In the Media library, upload the image if needed, then select the corresponding image and click Insert.
  16. Navigate through each sheet in edit mode and add the corresponding thumbnail. Then navigate to the App overview and review the result.

Example

Click on the Solution image to open as a pop-up, or right-click on the image for more options. To close, click on the image again.
