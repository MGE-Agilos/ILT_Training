Create Visualizations with Qlik Sense

Activity 4b: Including customer analysis sheets

Summary:

  • Explore and work with the Qlik Sense visualizations and other dashboard objects.
  • Represent data with other visualizations and tabular format.

Module: Creating visual analytics

Qlik Sense app: ABC Sales Analysis v1

Challenge

Create two new sheets in the ABC Sales Analysis v1 app to focus the analysis on customers.

Users should be able to make selections in these sheets using the fields: Year, Quarter, Division, Country, and Customer. In the first sheet, these fields will be presented as check boxes.

Create one button in each sheet to navigate to the other sheet, so both sheets can be reached without using the default sheet navigation button.

The first sheet, called Customer Sales, should include visualizations to address the following:

  • Show the Discount and Sales correlation per Customer, additionally, display the Num Orders and add a target for the Discount of 5%.
  • Compare Sales CYTD - LYTD by Customer using a color code chart.
  • Show customer details: Customer, Product Drill-down, Discount, Margin%, Sales, Sales trend*, Sales CYTD - LYTD, and Last order date**.

*Sales trend is the representation of the sales amount by Year or YearMonth as a sparkline. Use the Sales measure and the YearMonth Drill-down dimension.

**Last order date is the latest date (most recent date) when the customer made an order.

The second sheet, called Customer Orders, should help answer the following business questions:

  • What is the monthly order volume trend by Division? Are there any regions that have not sold any products?
  • What is the distribution and range of the order volumes per region, division, category, and product by customer? Use a Product drill-down dimension to be able to see an overall summary per region, division, product category, and product.
  • What are the exact figures of Num Orders and Num Orders CYTD for those customers? You should be able to see the summary and also expand the figures to see the breakdown per region, division, category, and product.

Analyze the new sheets. Select the current year and use the associative insights to find out more about the data not related with the selections. How many customers have not bought anything this current year? What products have not been sold?

Note: Measures and data shown depend on today's date. If you reload your app data, your results might differ from the solution image results.

Hints

  • Use the more suitable visualization per each item:

    • A scatter plot can be used when displaying the correlation between two measures.

    • Bar charts are good when comparing the same measure and you can color code the bars.
    • Tables can contain several columns and represent the data in detail. If you want to display breakdowns with summary values, pivot tables can help you achieve this functionality.
    • Line charts are effective at displaying trends over time.
    • Box plots are good when you need to compare range and distribution for groups.
  • For the Last order date measure, use the most recent date in the OrderDate field: Max(OrderDate).
  • Use the Selection tool and the associative insights to answer some of the business questions.

 

Step-by-step guide
  1. In the ABC Sales Analysis v1 app, create a new sheet with the title Customer Sales and open it in edit mode.

    2. Adding filters

  2. Add a Filter pane from the Assets panel (Charts) and include these dimensions in it: Year, Quarter, Division, Country.
  3. Adjust the filter pane so that it takes up the left side in the grid, leaving some free space at the bottom.
  4. Click over the Year field. Under Presentation, be sure Compact view, Checkbox mode and Histogram are enabled.
  5. Click over the Quarter field. Under Presentation, be sure Compact view and Checkbox mode are enabled.
  6. Click over the Division field. Under Presentation, be sure Compact view and Checkbox mode are enabled.
  7. Click over the Country field. Under Presentation, be sure Compact view and Checkbox mode are enabled. In the Show frequency option, select Relative frequency (%). Note that each Country will have an associate percentage in the list box, that represents the number of customers that are from that country divided by the total number of unique customers, multiply per 100. This is because the Country field comes from the Customers table in the data model.
  8. Add a Filter pane from the Assets panel (Charts) and include Customer as a dimension.
  9. Click over the Customer field. Choose Wildcard as Search mode. Under Presentation, be sure Compact view and Checkbox mode are enabled. In the Show data in option, select Grid as we want to visualize this list box in a few columns.

    10. Discount and Sales correlation (Scatter plot)

  10. Add a Scatter plot on the right side of the filter pane. Use Customer as the dimension, and Sales and Discount as measures.
  11. Add a third measure, Num Orders. This measure determines the bubble size.

  12. Using the properties panel, under Appearance for both the X-axis and Y-axis, set the Range to Custom.

  13. Resize the chart to leave some free space on the right. Include a title and subtitle in the chart.

    14. Sales CYTD - LYTD comparison (Bar chart)

  14. Add a Bar chart on the sheet. Use Customer as the dimension, and Sales CYTD - LYTD as the measure.
  15. In the Properties panel under Sorting, ensure that Customer is first.
  16. Under Appearance, set Colors to Custom by measure, turn Library colors off, and select Diverging gradient and Reverse colors. Then set Show Legend to Off.
  17. Add a chart title, CYTD - LYTD Sales Comparison.

    18. Customer details (Table)

  18. Resize the objects on the sheet to leave some free space at the bottom of the sheet and add a Table (from the Assets panel > Charts).
  19. Using the Properties panel, in the Data section, add the following dimensions and measures:
    • Customer as a dimension
    • Product Drill-down as a dimension
    • Discount as a measure
    • Margin% as a measure
    • Sales as a measure
    • Sales CYTD - LYTD as a measure
  20. Duplicate the Sales measure in the properties panel and unlink the new measure to apply the following changes to the measure:
    • Format the measure to Money and add the following Label: Sales trend.
    • Select None for Total functions.
    • Select Mini chart as Representation with Dimension as YearMonth Drill-down and Type as Sparkline. Additionally, click Colors to define the color of the sparkline and highlight the maximum and minimum points in the trend.
  21. Add another measure, Last order placed. For the last order date, we can use the most recent date in the OrderDate field, creating the following expression: Max(OrderDate).

  22. Apply a proper number format to the measures as follows:

    • Discount is formatted as Number (12.3%).
    • Sales CYTD - LYTD is formatted as Money.
    • Sales is formatted as Money. It needs to be displayed in $1000. In order to apply this format, you need to unlink the measure and use the following expression: [Sales]/1000Then format the measure to Money and add the following Label: Sales ($k).
    Note: To be able to format the measures that are using master measures, make sure you turn off the Master measure formatting property first.
  23. Under Appearance > Presentation > Styling > Chart, format the font of the headers to 14 points in size and use the green font color, then reduce the rest of the table font to 10 points.
  24. Show totals and position them at the bottom of the table.
  25. Add a title to this table, Customer sales details.
    Note: The table displays zero values and null values for some regions (note there are some customers with $0 sales). These customers are not buying any products at the moment and you can hide those from the properties panel using the Limitations (Exact value > 0 ) and clear Include null values in the dimension of the table. Then under the Add-Ons > Data handling, clear the Include zero values setting.

    26. Navigation button

  26. From the Assets panel > Charts, drag and drop a Button object into the sheet and locate the button at the bottom of the sheet, right below the filters. Then, go to Actions and navigation section > Navigation and choose Go to next sheet.
  27. Replace the text in the button to: Customer orders >> and change the background color and text color, style the button and include an icon.

    28. Customer orders sheet

  28. Duplicate the Customer Sales sheet and rename it as Customer Orders. Locate this new sheet right after the Customer Sales sheet.
  29. Replace the text in the button to: Customer sales >> and change the navigation to the previous sheet.
  30. Drag a Line chart from the Assets panel on top of the existing bar chart and select Replace with: Line chart.
  31. Use Month and Division as dimensions, and Num Orders as measure.
  32. Make sure the chart is sorted by Month. Turn off the Auto sorting in this dimension, and sort Numerically ascending. This way the months will be sorted from Jan to Dec.
  33. Under Add-ons > Data handling, clear Include zero values.
  34. Under Appearance > Presentation, select Area and Stacked Area. Then select Show data points, and set Value labels to Auto.
  35. Under Appearance > Colors and legend, set the legend position to Top.
  36. Add a title to the chart, Order volume trend.
  37. Delete the scatter plot and drag a Box plot into the free space in the sheet.
  38. Add Customer as dimension, Num Orders as measure, and Product Drill-down as second dimension. Then clear Include null values in the properties panel for this dimension.
  39. Under Appearance > Presentation section of the properties, set the boxes to horizontal. Then, under Colors, set the colors to single color (#8e477d).
  40. Add a title, Order volume range and distribution, and a subtitle to clarify the dimension is a drill-down dimension.
  41. Edit the existing table to change the title to Customer order details, and delete all the existing measures. Then add two new measures: Num Orders and Num Orders CYTD.
  42. Drag a Pivot table into the existing table and select Convert to: Pivot table. Then in analysis mode, move the Region title block in the table in front of Customer, so this dimension is located first in the table.

    43. Analyzing the sheet (Demo)

  43. Select P next to the Americas value in the pivot table to display the customers. Then select Americas using the box plot chart.
  44. Notice how the table drills down to display the North America and South America Divisions. What can you see in the box plot chart for North America? The Corner Store is a customer with an order volume that exceeds the normal average rate.
  45. Select North America and review the categories in the box plot. Hover over the outliers to see how The Corner store makes the highest order volumes in all of them.
  46. Navigate to the previous sheet, Customer Sales, using the button and notice where the same customer is located in the scatter plot. Hover over the bubble to see details about the Discount% for this customer.
  47. Select Sportswear in the CategoryName column in the table.
  48. Sort the Customer sales details table by Discount and identify which customer and which product got the highest average discount (25%).
  49. Use the Selections bar to move to the previous selection. Notice what happens to the Product column in the table.
  50. Use the Selections bar again to move to the next selection. Then clear all selections.
  51. Select the current year using the filter pane and then open the Selection tool by clicking Selections. Use the right panel (Associative insights) and click Generate insights. Then analyze the non-related data results for Num Orders measure using different dimensions.
