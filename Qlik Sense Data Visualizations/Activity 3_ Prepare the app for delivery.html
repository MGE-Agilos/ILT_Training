Create Visualizations with Qlik Sense

Activity 3: Prepare the app for delivery

Summary:

  • Understand how to review and prepare the app for handing over to consumers.
  • Work with master items, descriptions, and tags.
  • Add items to Favorites and monitor charts in the Qlik Cloud hub.

Module: Sharing insights, monitoring and finalizing the app

Qlik Sense app: ABC Sales Dashboard v2

Challenge

Prepare the ABC Sales Dashboard v2 app to hand over to business users. To accomplish that, do the following:

  • Delete any unused sheets and clean up any unused visualizations or objects.
  • Make sure every sheet is clear and all the visualizations are easy to understand. You can always include a sheet description, and use the titles, subtitles, and footnotes in visualizations. Showing details in charts can also help users to understand the visualization.
  • Add more information in descriptions and tags in the existing master measures and dimensions, if needed. For instance, add a description in the COGS master measure.
  • If you are working with Qlik Sense cloud editions, monitor some charts in the Hub.
  • Show details in the following charts, create master visualizations based on them, and include descriptions and tags:
    • Combo chart in the Dashboard sheet
    • Scatter plot in the Product analysis sheet

Hints

  • Add comments and labels to master dimensions and measures, using the Description and Tag areas when editing the master items.
  • The Show details option can be found in the visualization properties panel, under the Appearance > General section.
  • Right-click on a visualization and select Add to master items to create a master visualization.

 

Step-by-step guide
  1. Open the ABC Dashboard sheet in the ABC Sales Dashboard v2 app, in edit mode.

    2. Add comments and information

  2. From the Assets panel, under Master items > Measures, edit the COGS master measure.
  3. Type Cost of goods sold as Description and add Cost as one of the Tags for this measure. Then Save the changes.

    4. Monitor visualizations in the Hub (only for Qlik Sense cloud editions)

  4. In analysis mode, select 2023, then right-click on the Sales KPI and select Monitor in hub. Add Total Sales 2023 as name to the visualization and click Monitor.
  5. Clear selections and right-click on the Sales and YTD Sales trend, then select Monitor in hub. Add Sales and Margin trend as name and monitor the visualization.
  6. Go to the Hub (under Home > Your charts section) and review the charts you are monitoring. Click on the Total Sales 2023 KPI to go into the app again.

    7. Add the app to Favorites (only for Qlik Sense cloud editions)

  7. As it is an app that we will share with others, it could be convenient to save it in Favorites. This way you will be able to find the app easily in the Hub. Click on the Star icon when hovering over the app tile.
  8. Go to Home in the Hub, and check that the app is under the Favorites section.

    9. Display the object details and create master visualizations

  9. In edit mode, select the existing combo chart and using the properties panel, under the Appearance > General section, set the Show details to Show.
  10. Right click on the combo chart and select Add to master items. Use Sales, YTD Sales, and Margin% trend as Name. Note that you can add description and tags.
  11. Click Add. The visualization is added to the master items list.
  12. In the Product analysis sheet, select the scatter plot, and use the properties panel to set the Show details to Show.
  13. Right-click on the scatter plot and select Add to master items.
  14. Type an informative name, description, and tags and click Add. Notice how the master visualizations are now linked visualizations and the properties panels are not available.
    Note: The master visualization can be edited from the Assets panel > Master items. You will need to unlink the visualizations in order to make changes to it.

Solution

Click on the Solution image to open as a pop-up, or right-click on the image for more options. To close, click on the image again.
