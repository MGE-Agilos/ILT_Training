Create Visualizations with Qlik Sense

Activity 1: Working with maps

Summary:

  • Explore and work with Qlik Sense visualizations.
  • Use the Insight Advisor and the Chart suggestions wizard to generate new visualizations in the app.
  • Create new visualizations using the Assets panel.

Module: Working with other visualizations and properties

Qlik Sense app: ABC Sales Analysis v2

Challenge

Analyze sales based on regions where customers are located (countries and cities) using a new sheet in the ABC Sales Analysis v2 app.

Create a Regional sales sheet in the app where users should be able to:

  • Identify where the customers are located and see the sales in each region (country and city). Use a map to analyze this:
    • Color the countries by Sales
    • Identify the cities with shapes (triangles) where the size and color of the shape represent the Sales measure
    • Show Sales, Num Orders, and Margin% information when hovering over a city
    • Display the country flag for each country and city in the tooltip*
    • Display the cities (city layer) only when the zoom level is 4x zoom or more
    • Display a dynamic subtitle to show the number of cities and customers

    • Additionally, users should be able to filter and make selections by Division, Country, City, and Customer using a filter pane on this sheet.

    *Using flag images provided online in an expression. The images are located in https://qt-lss.s3.amazonaws.com/Curriculum/Country-Flags/ folder. Each flag has the name of the country and a png extension.

    Bonus point:

  • Analyze the exact figures of Sales, Num Orders, and Num Customers by each country and city
    • Use a tabular representation and style text and headers
    • Highlight the rows when hovering over them

Hints

  • Use the following visualizations in the new sheet:
    • Sales map with layers: Country (Area layer) and City (Point layer)
    • Sales, Num Orders, and Num Customers by month (Pivot table)
  • In the map:
    • Use the Options > Layer display section in the properties panel of each layer

    • Add a subtitle using text and count() functions; e.g. to display the number of cities:

      count(distinct City) & ' cities'

 

Step-by-step guide
  1. In the ABC Sales Analysis v2 app, create a new sheet, Regional Sales, and open it in edit mode.

    2. Note: You may need to use the Advanced options toggle to access the Assets panel in Edit mode.

  2. Drag the Customer field onto the sheet with the Shift key pressed to create a filter pane. Then locate it on the left of the sheet.

    3. Sales Map

  3. From the Assets panel, add a Map onto the sheet next to the Customer filter and resize it to make it bigger.
  4. In the properties panel, expand the Map settings and review the options for Base map, Projection, Map language, Units, and Selection method. Then select Pale for Base map and Adaptive projection.
  5. Under Appearance > General, add a title Sales map.
  6. Under Layers, click Add layer and select Area layer. Then under Dimensions, select Add and choose Country.
  7. For this new layer, under Colors, set the Colors to Custom and select By measure in the drop-down menu. Then choose the Sales measure and the Sequential gradient for the color scheme.
  8. Under Options > Tooltip, select Custom. Then under Images, select URL as Type and enter the following expressions as URL:

    ='https://qt-lss.s3.amazonaws.com/Curriculum/Country-Flags/'&Country&'.png'

    Note: In Cloud editions of Qlik Sense external URLs and content access must be enabled in the Management Console, under Content Security Policy.
  9. Click on the Country Area layer (at the top of the properties panel) to go back to the Sales map properties. Then click Add layer again, and choose a Point layer this time.
  10. For this new point layer, under Dimensions, click Add and select City. Then under Size & Shape, select Size by Sales and Triangle as Shape. Notice you can also use a custom image instead of a shape.
  11. Under Colors, make sure that the Colors are set to Custom and select By measure in the drop down menu. Choose Sales for measure and for the Outline color select a bright red color from the palette.
  12. Under Options > General, turn Show labels on. Then, under Options > Tooltip, add two measures: Num Orders and Margin%. You can also add in this layer the URL Images for the country flags, as in step 8.
    13. Note: Under the Options > Layer display properties menu of each layer, a calculation condition can be specified to hide or show the layer. In the Options > General properties, you can also select Disable selections to avoid selections being made at this layer.

  13. Under Options > Layer display (for the City Point layer properties panel), set the Show in zoom levels property to Custom, and use the scroll to initial zoom to 4x zoom.

  14. Click on the City Point layer (at the top of the properties panel) to go back to the Sales map properties.

  15. Under Appearance > General, add the following 3 expression as the Subtitle:

    =count(distinct Customer) & ' customers in ' &count(distinct City) & ' cities'

  16. Notice there is an info icon on the map. Click on the info icon to see the warning message. The data quality of the City field in this data set might not be good enough for Data manager to identify all the city points in the map. In a real scenario, you need to consider these messages, as data that is not located will be missing from the map. In these cases, it is always recommended to support the visualizations with a table or other visualizations where you can investigate those details.

    17. Bonus point:

    Pivot table

  17. Add a Pivot table on the sheet. Use Country and City as dimension Rows, using the properties panel.
  18. Add Sales and Num Orders as Measures. Then, format Sales using the Money format, disabling the Master measure format, as data that is not located will be missing from the map.
  19. Add a third measure using the following 3 expression:

    count(distinct CustomerID)

  20. Label the new measure as Num Customers.
  21. In the properties panel, under Add-ons > Data handling, clear Include zero values.
  22. Under Appearance > Presentation, clear Indent rows. Then, select Styling > Chart and apply the following changes:
    • Set the Header font size to 14 and use the dark green for the color.
    • Set the Content font size to 14.
    • Set Highlight rows on hover to on and select yellow as the color for the Row hover color.

    23. Analyze regional sales

  23. In edit mode, activate the Navigation controls in the map from the properties panel, under Appearance > Presentation, set Navigation to On and Location finder to On.
  24. In analysis mode, use the Navigation controls on the map to zoom in and review Europe, then select the country with the higher sales using the map and compare your results with the solution image. See how the selection affects the map. Notice when the city information is displayed in the map.
  25. Use the Circle selection to see how you can select customers and cities within a 250km radius from Berlin.
  26. Clear selections and use the location finder (Find my location icon) to locate your current location in the map. Then zoom out to see the whole country. Note that if you are using a virtual platform, your location might display as the location of the virtual server.
Note: Using the map exploration menu, you can: toggle layers, define size objects, change base map, change map language, etc.

Solution

Click on the Solution image to open as a pop-up, or right-click on the image for more options. To close, click on the image again.
