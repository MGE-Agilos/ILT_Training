Create Visualizations with Qlik Sense

Challenge 2: Working with extension bundles

Summary:

  • Explore and work with the Qlik Sense extensions.
  • Use visualizations and objects in the Dashboard and Visualization extension bundles.
  • Configure extension properties using the properties panel.

Module: Working with other visualizations and properties

Qlik Sense app: ABC Sales Analysis v2

Challenge

Create a sheet in the ABC Sales Analysis v2 called Extensions and add the following objects:

  • The Word Cloud chart extension with Product category as the dimension and Sales as the measure - use the Appearance section in the properties to customize this extension as follows:
    • Change the orientation of the text to horizontal, setting Start Angle to 0
    • Change the Font Min Size to 12 and the Font Max Size to 40
    • Change the font color to set a color range using Enable Custom Range
  • The Radar chart extension with Quarter and Product category as the dimensions and the Sales as the master measure.
  • The Trellis container extension with Division as the dimension and the Sales trend as the master visualization (this line chart is available as a master visualization in the app).
  • The Straight table extension with Division, Country, Product category, Product and Customer as the dimensions and Sales, Margin, Discount, Num Customers and Num Orders as the measures. Enable the Chart exploration panel.

Add the following extensions from the Qlik Dashboard bundle:

  • Date picker - select from the OrderDate field, a range of dates using the calendar format.
  • Animator - select the Week field to animate all the visualizations in the sheet by this field.
  • NL Insights - select Sales as the measure and Division and CategoryName as the dimensions.

Qlik Sense Enterprise / Cloud editions of Qlik Sense: These extensions have already been included into your environment and they are available under the Custom objects section in the Assets panel. Select Qlik Dashboard bundle or Qlik Visualization bundle to see all the related objects.

Qlik Sense Desktop: The Qlik Extension bundles are installed during the QSD installation process if you selected that option. If you did not include the bundles when you installed QSD, you can still import these extensions:

  1. Download the extension bundle files (available in the Student files) on your computer.
  2. Using the File Explorer, navigate to C:\Users\<username>\Documents\Qlik\Sense\Extensions folder.
  3. Unzip the extension bundle files you downloaded, extracting them in the Extensions folder. The extensions are now available in Qlik Sense Desktop.

Then make selections in the Date Picker extension and the other visualizations to analyze the data to answer the questions below:

  • How were the sales in each division for the period: 4th Jan 2023 - 31th March 2023? (Use the Date picker to select the dates)
  • What are the top 3 selling product categories for the same period? (Hover over the Radar chart to see the sales values)
  • In the Baby Clothes and Swimwear categories, how were the sales in the different divisions in this period? (You can make selections in the Word cloud chart)
  • What is the full trend of each division over all the years for Swimwear products? What quarter is the strongest overall for this category? (Use the Word cloud chart to make the product category selection, use the Radar chart to see what quarter is the strongest over all regions, then use the Trellis container to review the trend and select the latest year in one of the trends to see the details per month of this year.)

To finish this challenge, play the animation in the sheet to see how the product categories trend over the weeks. Then discuss other interesting uses for the Animator extension. Remember to clear the selections once you finish working with this extension.

Note: The objective of this challenge is to test whether you feel comfortable with the concepts that have been discussed so far in your class. There is no Step-by-Step section in this activity, so you must only rely on the Hints below to understand the tasks you have to execute.
As you are still new to many of these concepts, please don't hesitate to reach out for more guidance (help file, instructor, Qlik Community).

Hints

  • From the Assets panel, under g Custom objects, you can add all the extensions to your sheet:
    • Under the Qlik Visualization bundle, you can add the Radar chart, Word Cloud Chart, Trellis Container and the Straight table.
    • Under the Qlik Dashboard bundle, you can add the Date picker, Animator and the NL Insights.
  • Use the properties panels to configure extensions as you do with other standard charts.
