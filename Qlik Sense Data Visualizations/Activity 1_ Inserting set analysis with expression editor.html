Activity 1: Inserting set analysis with expression editor

Summary:

  • Use the expression editor panel to define set analysis based on selections.
  • Avoid typing the expression.
  • Use set expressions out of aggregations.

Module: Controlling data selections

Qlik Sense app: ABC Sales Dashboard v1

Challenge

Create a new KPI object in the sheet Sales Analysis, in the ABC Sales Dashboard v1 app.

  • The object will display the Americas Avg deal size.

  • The expression will not be affected by user selections.

Note: Measures and data shown depend on today's date. If you reload your app data, your results might differ from the solution image results.

Hints

  • Copy and paste the "Avg deal size*", as this is very similar to the object we want achieve. Then, modify the expression.
  • Note that it is important to place the cursor before the expression once you are going to inset the set analysis.

 

Step-by-step guide
  1. In the ABC Sales Dashboard v1 app, go to the sheet called Sales Analysis and click Edit sheet.
  2. Copy the "Avg deal size*" object and paste it below the "Current Year H1" button.
  3. Come back to analysis mode and select the "Americas" button on the left to apply this selection.
  4. When the selection is applied to the app, under Data>Measures>Avg deal size, click the button to Unlink measure.
  5. Open expression editor and look at the right side of the window. Under Set expressions, select Use current selections and click Insert. Confirm that the set expression has been inserted ahead of the existing expression:
    {<Division={'North America','South America'}>}[Sales]/[Num Orders]

    We are using an outer set analysis, as the set expression is outside the aggregations, because master measures are applied in the expression. It is equivalent to this inner set analysis:

    Sum({<Division={'North America','South America'}>}Sales)/Count({<Division={'North America','South America'}>}distinct OrderID)
  6. Click Apply and Edit sheet. Note that the result of the KPI object that we have just created is the same as the other above, because we applied the Americas button selection to create the set expression.
  7. If you clear selections, you will confirm that the Avg deal size for America is a little bit lower that the result displayed in the first KPI.

Solution

Click on the Solution image to open as a pop-up, or right-click on the image for more options. To close, click on the image again.
