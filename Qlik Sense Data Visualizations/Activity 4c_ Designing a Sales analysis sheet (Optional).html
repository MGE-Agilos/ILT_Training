Create Visualizations with Qlik Sense

Activity 4c: Designing a Sales analysis sheet (Optional)

Summary:

  • Explore containers and how we can organize visualizations in the sheet.
  • Work with other charts and pivot tables and explore their properties.

Module: Creating visual analytics

Qlik Sense app: ABC Sales Analysis v1

Challenge

Create a new sheet called Sales analysis in the ABC Sales Analysis v1 app.

Then add the following information:

  • Sales and Margin% comparison per Customer
  • Sales and Margin composition, including Gross sales*, Discount amount**, Net Sales***, COGS and Margin
  • Sales composition based on Sales type and Product category
  • Sales YTD versus Year sales target**** by Sales person comparing with the previous year sales

Use the same space in the sheet for the first two visualizations, and let the user switch between them and select which one to visualize.

*Gross sales is calculated as the sum of Quantity multiplied by UnitPrice.

**Discount amount is calculated as the Gross sales multiplied by the Discount percentage (Discount field).

***Net Sales is the Sales master measure created in the app.

****Year Sales Target is the sales target fixed for each sales person. This target is fixed to double the previous year sales (Sales LYTD).

Use a Combo chart, a Waterfall chart, a Mekko chart, and a Bullet chart, and include all of them in a Container.

Also, display the Divisions and Customers Sales and Margin% details over all the years and quarters in a Pivot table. The table should show the following:

  • Summary view: Sales and Margin% by Division and Year
  • Detailed view: Sales performance based on Customers in each Division over the years and quarters

When the pivot table is configured, download or export the pivot table data to Excel.

Bonus Point

Include a Line chart in the container using the Year Drill-down dimension to display the following information:

  • Sales in the Year or MonthYear and the linear and average trend lines of the displayed sales in the trend.
  • Sales difference with the previous period in the trend and the average of the differences between periods.

Hints

  • In a Combo chart, each measure can be represented in the primary or secondary axis and it can use different styles (Bar, Line, and Marker). Measures with different value ranges can be displayed in different axes and different styles.
  • Waterfall charts only contain measures. Each new measure added in the chart can Add or Subtract from the previous measure, or display a Subtotal.
  • In a Mekko chart, we can compare groups, while being able to get the details of each category item contained within these groups.
  • In a Bullet chart, we can display a measure versus its target by different category items. This allows a double comparison toward its own individual target and towards other category items. Additionally this chart also can display segments to include another level of comparison. In this case you can compare Sales YTD versus Sales LYTD using two segments.
  • In analysis mode, Pivot tables allow users to regroup the data by moving dimensions and measures between rows and columns. Users can also expand or collapse the information in these tables.
  • You can right-click on the visualization and select the download option to export it as data, image or PDF.

 

Step-by-step guide
  1. In the ABC Sales Analysis v1 app, create a new sheet called Sales analysis. Then open the new sheet in edit mode.

    2. Combo chart

  2. Add a Combo chart to the sheet. Use Customer as the dimension and Sales as the measure.
  3. Add a second measure, Margin%, using the properties panel. After expanding the measure, click the More properties button and, under Presentation, select Marker and Secondary axis to display it.
  4. Add a title, Sales vs Margin Customer comparison. Then resize the combo chart to leave some free space on the right.

    5. Waterfall chart

  5. Add a Waterfall chart in the sheet, next to the combo chart.
  6. Click Add measure, and then use the 3 Expression editor to type:
    Sum(Quantity * UnitPrice)
  7. Using the properties panel, under Data, label the measure as Gross Sales and ensure the Measure operation is Add.
  8. Add three more measures with the following settings:

    LabelMeasure ExpressionOperationCheck Subtotals
    DiscountSum(Quantity * UnitPrice * Discount)SubtractNo
    SalesUse the Sales master measureSubtotalsNo
    COGSUse the COGS master measureSubtractYes (Subtotal Label: Margin)

  9. Add a title, Sales and Margin composition.

    10. Mekko chart

  10. Add a Mekko chart in the sheet with Product category as column dimension, Sales type as cell dimensions and Sales as measure.
  11. Add a title, Sales composition by Sales type and Category.

    12. Container for the visualizations

  12. Add a Container from the Assets panel > Charts. The container is added into the sheet. You can add content from master visualizations, or by dragging and dropping existing charts in the sheet into the container. By clicking Add content you can also create new visualizations.
  13. Drag the existing combo, mekko, and waterfall charts into the container. Now they are embedded into the container instead of being single objects. Note that you can still edit their properties from the container (under Content).
  14. Locate the container at the top of the sheet and resize it to full width. In the container properties, under Container, select Show icons.

    15. Bullet chart

  15. In the Container properties, click Add and then select the Bullet chart.
  16. Select Sales person as a dimension and Sales CYTD as measure. Then in the measure, set the target to twice the Sales LYTD with following expression:

    [Sales LYTD]*2

  17. Set Use segments to On and add the =[Sales LYTD] expression as the limit. Make sure the first segment is set to gray color and the second segment is set to white.
  18. Under Appearance > Presentation, select Horizontal orientation. Then under Appearance > General, set the title to Sales person target sales.

    19. Bonus Point: Line chart

  19. Add a Line chart in the sheet and use Year Drill-down as a dimension and Sales as a measure.
  20. Duplicate the Sales measure, unlink it and rename the measure to Sales Diff vs Previous. Then apply the Difference modifier and include the average trend line.
  21. For the Sales measure, add two trend lines to the measure using the Linear and Average trends. For the linear trend, removed the Dashed style and set the color to orange.
  22. Style the lines and show data points.
  23. Add a title, Sales and Sales difference. Include subtitles and footnotes to explain the trends and the drill down dimension.

    Then, drag the line chart onto the container to add it into this object.

    24. Pivot table: Summary view

  24. Add a Pivot table in the sheet and use Customer as a dimension.
  25. From the Properties panel, add Division as a new Row and add Year and Quarter as new Columns.
  26. There is no need to show the Division column in the table if only one selection is made. Expand the Division tab in the properties panel and under Show column if type:
    = GetSelectedCount(Division) <> 1

    This means that the column is hidden if the user has selected only one division and it appears in all other cases, to allow the user to select more values or analyze data.
  27. Clear the Include null values option for all dimensions.
  28. Move the Division dimension on top of the Customer dimension under Row.
  29. Add Sales and Margin% as two new Measures. Then, format the Sales measure to Money.
  30. Disable edit mode and resize the first column in the pivot table, reducing the width in the column. Enable edit mode again to apply these changes.

    31. Pivot table: Detailed view

  31. With edit mode disabled, move Division and Customer from row to column, to the left of Values in the table.
  32. Move Year and Quarter from column to row. Each Division can then be expanded by clicking P to show the customers. Each Customer appears as a column and has its own Sales and Margin% columns below. On the left, Year shows as the top level in the rows.
  33. Expand the last year and see the quarters.

    34. Export to Excel

  34. Right-click the chart object and select Download > Data. Then click the Download button.
    Note: In QSEoW and QSD, right-click the chart object and select Download as... > Data, then click the link to download the data file.
  35. Review the Excel file from your default downloads location.
