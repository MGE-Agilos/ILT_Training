Create Visualizations with Qlik Sense

Activity 4: Working with bookmarks

Summary:

  • Explore and work with bookmarks in a Qlik Sense app.
  • Understand how to apply bookmarks and change them.
  • Set a default bookmark in the app.

Module: Overview, selecting and searching data

Qlik Sense app: Vintage movies

Challenge

In the Vintage movies app, create a bookmark, 1990TopMovies, that will take you to the Movie Selector sheet with the following selections:

  • Decade — 1990's
  • Rating — 4 and 5

Then create a second bookmark, 1990BadMovies, with the same behavior but selecting 1990's and 1.

Hints

  • Bookmarks can be created from a sheet in analysis mode and when applied, they take you to the sheet they were created from.
  • You can use the search or the selection panel to make the selections required in the Decade and Rating fields.
  • In order to test the bookmark's navigation, you can apply the bookmark from another sheet in the app.

 

Step-by-step guide

  1. Open the Movie Selector sheet in the Vintage movies app, and clear all selections, if necessary.

  2. Use the search box to find the following values and then select:

    1. Decade—1990's

    2. Rating—4,5

  3. Using Bookmarks in the toolbar, click Create new bookmark, and give the bookmark a name: 1990TopMovies.
  4. Change the selections in the Rating field to select 1.
  5. Using Bookmarks in the toolbar, create a second bookmark called 1990BadMovies.
  6. Clear all selections and navigate to the Dashboard sheet.
  7. Using Bookmarks in the toolbar, select the 1990TopMovies bookmark. The Movie selector sheet opens with the predefined selections in Decade, and Rating.
  8. Using the Bookmarks button again, right-click on the 1990TopMovies bookmark and select Set as default bookmark. This will set the bookmark as default for the app. The selections of this bookmark will be applied to the app when the user opens the app.
