Data Modeling for Qlik Sense

Activity 2: Navigate within an app

Summary:

  • Switch between app sheets.
  • Explore bookmarks.
  • Use a bookmark to navigate to a pre-defined selection.

Module: Modeling data with Qlik Sense

Qlik Sense app: Vintage Movies

Challenge

Explore the Vintage Movies app and familiarize yourself with the user interface by doing the following:

  • Use the Navigation menu to locate the view which will allow you to verify when the data was last loaded.
  • Navigate through the sheets and explore their contents.
  • Use the App overview to change the view mode to List view.
  • Identify which bookmarks are currently available in this app.

Hints

  • Sheets, bookmarks, and stories can be accessed from both the App overview and Sheet interface.
  • Select Movies from 1980's Rating 5 bookmark.

 

Step-by-step guide
  1. Re-open the Vintage Movies app, if required, and navigate to the App overview, as necessary.
  2. Locate and open the Dashboard sheet. This is one way you can navigate to sheets in the app.
  3. Navigate to the Movie Selector sheet using the Sheets icon from the toolbar. This is another way you can navigate to sheets in the app.
  4. Navigate to the next sheet in the app using the arrow buttons from the toolbar. This is a way to navigate consecutively forwards or backwards through the sheets.
  5. Check if you can edit the sheet. If editing is allowed, enable the edit mode in the sheet.
  6. If you are in edit mode, navigate through the editing options, then exit edit mode.
  7. Explore other buttons on the toolbar; hover over each button to see the tool tips pop-up.
  8. Navigate back to the App overview using the Navigation menu.
  9. Change the view mode from Grid to List view. All the sheets appear in a list format.
  10. Navigate to the Bookmarks view. The app contains one bookmark, Movies from 1980's Rating 5.
  11. View the bookmark details by clicking the Details icon. The information provided contains a description, fields in which selections were made, and the sheet that will be activated when the bookmark is selected.
  12. Close the bookmark details panel by clicking the icon a second time.
  13. Select the bookmark to open the sheet. The Movie Selector sheet appears with selections in Rating and Decade. Compare the sheet with the solution image.

Solution

Click on the Solution image to open as a pop-up, or right-click on the image for more options. To close, click on the image again.
