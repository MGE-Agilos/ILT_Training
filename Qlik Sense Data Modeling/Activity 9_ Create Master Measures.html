Data Modeling for Qlik Sense

Activity 9: Create Master Measures

Summary:

  • Create Master Measures for use in analysis mode.
  • Leverage the Master Measures in a table object.

Module: Applying finishing touches

Qlik Sense app: ABC Sales Data

Challenge

In the ABC Sales Data app, create two Master Measures using simple aggregation functions. To achieve this, complete the following tasks:

  • From the Data model viewer, preview the LineSalesAmount field and Add as measure.
  • Replace the Expression withSum(LineSalesAmount). Add some Tags and a brief Description. Use Sales as the new measure Name.
  • Add a second Master Measure named COGS, with Sum(CostOfGoodsSold) as the Expression.
  • In the Master Items sheet, add the new Master Measure Sales and replace the existing Sales column.
  • Add COGS as a column in the same table.
  • For both Master Measures, change the Number formatting to Money.

Hints

  • Once a Master Measure is created, it can be reused multiple times in the app.
  • Modifying the expression of a Master Measure will update all the objects that use this measure.

 

Step-by-step guide
  1. In the Data model viewer, select the LineSalesAmount field from the OrderDetails table.
  2. Preview the table and select the button to Add as measure.
  3. Change the Master Measure Expression to the following:
    Sum(LineSalesAmount)
  4. Set the Name to Sales and add the following Description: Sales value at order line level.
  5. Set the Number formatting to Money.
    6. Note: When using Money as the Number formatting, Qlik Sense utilizes the settings from the data load script to determine the format pattern. If required, it can be overridden.
  6. Add four Tags: Amount, Line, Order, and Sales.
  7. Create the Master Measure.
  8. Repeat the process for the CostOfGoodsSold field, from the OrderDetails table: Select the button to Add as measure and change the Expression to the following:
    Sum(CostOfGoodsSold)
  9. Set the Name to COGS and add the following description: Cost of goods sold at order line level.
  10. Set the Number formatting to Money.
  11. Add four tags: Cost, Line, Order, and Goods. Then, click the Create button.
  12. Navigate into Analyze mode and open the Master Items sheet.
  13. Enable the edit mode and select the table object you created previously.
  14. From the Assets panel, locate the Sales Master Measure. Double-click to replace the existing Sales column. Notice that you are not prompted to choose an aggregation function. This is because the Master Measure includes the Sum function.
  15. Double-click the COGS Master Measure to add it to the table object.
  16. Disable the edit mode and notice that the table now contains three columns that are all Master items in the library. The use of these fields in other objects means they are all based on the same definition.
    Note: You can change a Master Dimension or Master Measure by selecting it from the Assets panel, in the Master items tab, and clicking Edit in the pop-up.
    The changes apply to all the visualizations that use the Master item.
