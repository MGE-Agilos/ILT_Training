Data Modeling for Qlik Sense

Activity 7 - Optional: Master Dimension from an expression

Summary:

  • Use the Data model viewer to generate a new Master Dimension, based on an expression of Quarter and Year.
  • Use the new Master Dimension to replace the existing dimension on a Line chart.

Module: Applying finishing touches

Qlik Sense app: ABC Sales Data

Challenge

In the ABC Sales Data app, create a calculated dimension and add it to the Master items library by doing the following:

  • In the Data model viewer, preview the Quarter field in the LinkTable and select the button to Add as dimension.
  • Use the expression =Quarter & '-' & Year and name the new Master Dimension Quarter Year. Add Tags and a Description.
  • Navigate to the Sales vs. Quota sheet and add the new Master Dimension, Quarter Year, as a Filter pane.
  • On the line chart, replace the Year dimension with Quarter Year.
  • Sort the line chart by expression. Use (Year * 10 + Right(Quarter, 1)) to fix the sorting issue.

Hints

  • The Quarter Year Master Dimension is based on the Quarter and Year fields.
  • Once the Quarter Year Master Dimension is created, it can be reused multiple times in the app.

 

Step-by-step guide
  1. In the Data model viewer, select the Quarter field in the LinkTable and preview the table.
  2. Select the button to Add as dimension.
  3. In Field, replace the current field with the following expression:
    =Quarter & '-' & Year
  4. Name the Master Dimension Quarter Year and set the Description to Quarter Year field displayed as QQ-YYYY.
  5. Add the following four Tags: Calendar, Date, Month, and Year.
  6. Click the create button, close the window,y and navigate into Analyze mode to open the Sales vs. Quota sheet.
  7. Enable the edit mode. Add the Quarter Year Master Dimension from the Assets panel, as a Filter pane on the sheet. Notice the data displayed in the format QQ-YYYY.
  8. Drag the Quarter Year Master Dimension on the line chart to replace the existing Year field. The chart dimension changes to Quarter Year.
  9. Review the line chart. Is there any problem with the new dimension? The order of the values is not correct in the time trend. You need to resolve the sorting issue.
  10. In the Properties panel, modify the Sorting options for the line chart as follows:
    1. Sort the chart by Quarter Year.
    2. Disable the Automatic sorting.
    3. Sort the chart by the following expression, ascending:

      Year * 10 + Right(Quarter, 1)

    11. Note: As there is no easy way to sort the new Quarter Year field, you have to Sort by expression and use the above formula. This formula multiplies the Year value by 10 and adds the Quarter number to it. So, the list of values it returns and uses for sorting is:
    ...20153, 20154, 20161, 20162...
  11. Stop editing the sheet and observe the results.
