Data Modeling for Qlik Sense

Activity 3 - Challenge: Load OrderDetails and Customers data

Summary:

  • Use the existing data connection to select and load data from the orderdetails and customers tables.
  • Load the orders table data and verify it is associated with orderdetails and customers tables.

Module: Sourcing and loading data

Qlik Sense app: ABC Sales Data

Challenge

In the ABC Sales Data app, load the orderdetails table in your data model by doing the following:

  • Navigate to the end of the Orders script section in the Data load editor.
  • Use the correct data connection to select data from the ABC_2025 database.
  • Load all the fields from the orderdetails table.
  • Capitalize and move the table label OrderDetails: above the LOAD statement.
  • Remove the superfluous LIB CONNECT TO 'ABC'; statement, as the connection is still open from loading the Orders table.
  • Navigate to the Customers script section and use the ABC data connection.
  • Load all the fields from the customers table.
  • Capitalize and move the table label Customers: above the LOAD statement.
  • Click the Load data button.
  • Verify that the three tables are associated, and explain why.
Note: The objective of this challenge is to test whether you feel comfortable with the concepts that have been discussed so far in your class. There is no Step-by-Step section in this activity, so you must only rely on the Hints below to understand the tasks you have to execute.
As you are still new to many of these concepts, please don't hesitate to reach out for more guidance (help file, instructor, Qlik Community).

Hints

  • The Select data dialog in the data connection can be used to load the tables.
  • Although both orderdetails and customers tables can be selected at the same time in the Data load editor, we want the two loading scripts to be placed in different sections.
