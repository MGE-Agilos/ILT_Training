Data Modeling for Qlik Sense

Activity 5: Managing script sections

Summary:

  • Create a new analytics app.
  • Modify app properties.
  • Create and arrange a plan of script sections.

Module: Modeling data with Qlik Sense

Qlik Sense app: ABC Sales Data

Challenge

Create a new analytics app named ABC Sales Data. In the new app, complete the following tasks:

  • Upload and use the SalesChart.jpg file, located in the student collateral.
    Qlik Sense on Windows: The required SalesChart.jpg image is already available for use in the Default library.
  • Add the following 3 script sections:
    • Orders
    • Products
    • Customers

From this activity onwards and throughout the rest of the course, you will use this app to develop new content, edits, etc. The app is going to be based on the ABC Apparel Inc. scenario.

Hints

  • New apps can be created in the hub.
  • Images can be added to the content library using the Upload media option.
  • Script sections can be added in the Data load editor.
Qlik Sense Desktop: Images located under %Documents%\Qlik\Sense\Content\Default are displayed in the Default content library.

 

Step-by-step guide
  1. At the hub, add a new analytics app and name it ABC Sales Data.
  2. Leave the app default settings and click the Create button to open the app for editing.
  3. Replace the default app thumbnail with the SalesChart.jpg image, provided with the course collateral. Make sure the image is in place before you execute this step.
    4. Qlik Sense Desktop: The image must be added into the %Documents%\Qlik\Sense\Content\Default folder.
    Qlik Sense on Windows: If the image does not exist in the Default content library, upload it from the C:\Qlik Sense Data\ABC Images using the Upload media feature. If Upload is not possible, use an image of your choice.
  4. Open the Data load editor.
  5. Create a new script section in the Sections area and name it Products.
  6. Add three additional script sections, and name them: Sales, Dates, and Customers.
  7. Drag to move Sales before Products, but after Main.
  8. Rename the Sales section to Orders.
  9. Delete the Dates section, as you do not need it at this time.

Solution

Click on the Solution image to open as a pop-up, or right-click on the image for more options. To close, click on the image again.
