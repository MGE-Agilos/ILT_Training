Data Modeling for Qlik Sense

Activity 8 - Optional: Qlik Sense with R Demonstration

Summary:

  • Explore the Qlik Sense Advanced Analytics Integration functionality.
  • Leverage the Qlik demos site.

Module: Exploring security and advanced concepts

Qlik Sense app: Demo site | UK Crime

Challenge

In the Qlik Demos website, review how R is utilized and how it works in an app.

  • Access the Qlik Sense demo site (https://demos.qlik.com/qliksense) and login with your Qlik Account.
  • Select the Advanced Analytics Integration tag and select the UK Crime Demo app.
  • Launch the Demo app and identify the sheets/charts where R is used.
  • Access Qlik Developer (https://qlik.dev/changelog/60-api-data-connections-dcaas) and review the information on Analytic Data Connections.

Hints

  • Charts that use R look very similar to Qlik Sense standard visualizations, but they are customized for a specific purpose.

 

Step-by-step guide
  1. In a browser window, access the Qlik Demos site (https://demos.qlik.com/qliksense). Login with your Qlik account credentials. (If you do not have a login, select Login from the top navigation and, in the login window, select Register.)
  2. Use the Tags drop-down to select Advanced Analytics Integration. This filters the demos that make use of the Advanced Analytics Toolbox.
  3. Select UK Crime. Then, click Launch Demo to review the app.
  4. Explore the different sheets in the app and identify the visualizations that contain Advanced Analytics expressions.
    Hint: Read the explanatory comments on every sheet.
  5. Navigate to the Crime Forecasts sheet and adjust the slider to increase or decrease the Forecast Months value. Observe the line chart adjusting accordingly.
  6. Navigate back to the App overview. Right-click and Duplicate the Crime Forecasts sheet. Open and edit the sheet you just created.
    7. Note: For the Duplicate option to appear when you right-click, make sure you are logged on with your Qlik account.
  7. From the Assets panel, navigate to the Master items and then, Measures. Observe the list of all the Master Measures. The line chart uses three of these Measures: HW Forecast, HW Lower 80% and HW Upper 80%. Click to see their expressions. They are all using R script.
    8. Note: The purpose of this lesson is to demonstrate how we can use R expressions from within Qlik Sense. However, the explanation of the R syntax and what they do is beyond the scope of this course.
  8. Navigate to the Clustering by Crime Type sheet and Duplicate it. In edit mode, review the dimensions and measures of the scatter plot. Notice that neither the dimensions nor the measures of the chart use R syntax. To achieve clustering the Crime Types using the K-means clustering algorithm, the R expression is used to color the data points.
  9. In the Properties panel, navigate to the Colors and legend section under Appearance. Notice that the Crime Percentage and Resolution Time Clusters Master Measure is used to color the data points. From the Assets panel, find the Master Measure and review its expression. R syntax is used to distinguish the crime types that belong to the same cluster by using the same color.
  10. The expression used in this Master Measure is dynamic. We can use the Clusters drop-down menu at the top right of the sheet to change the number of clusters required for our analysis. Navigate to the App overview window to stop editing the sheet.
  11. Similarly, navigate to the other app sheets and identify other objects with Advanced Analytics expressions.
  12. In a new browser window, access the Qlik Dev site (https://qlik.dev/changelog/60-api-data-connections-dcaas).
  13. Use the links within the page to navigate to and learn more about analytic data connections.
    14. Note: These solutions are provided by Qlik developers free of charge. Download them from the Qlik Sense Demos site and start using them in your app.
