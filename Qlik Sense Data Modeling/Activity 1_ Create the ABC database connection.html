Data Modeling for Qlik Sense

Activity 1: Create the ABC database connection

Summary:

  • Create a data connection to an Azure SQL database.
  • Test the connection.

Module: Sourcing and loading data

Qlik Sense app: ABC Sales Data

Challenge

In the ABC Sales Data app, create a new connection with the following properties:

  • Connection: Azure SQL Database
  • Server: qlik-es.database.windows.net
  • Port: 1433
  • Database: ABC_2025
  • User name: QlikEducationStudent
  • Password: AzureTraining1
  • Name: ABC

Hints

  • New connections can be created from the Data load editor.
  • The property values need to be added manually.
  • Testing the connection is always a wise choice.

 

Step-by-step guide
  1. In the Data load editor, create a new data connection.
  2. Select Azure SQL Database from the provider list.
  3. Enter the Server value of: qlik-es.database.windows.net.
  4. Ensure the Port is set to: 1433.
  5. Enter the Database name of: ABC_2025.
  6. For the User name, use: QlikEducationStudent.
  7. Provide a Password of: AzureTraining1.
  8. Edit the default-generated Name (bottom most field in the pop-up window) to: ABC.
  9. Test the connection and make sure it is successful.
  10. Create the connection.
  11. After you create the connection, confirm it appears in the Data connections panel.
    Qlik Sense on Windows: The connection name also contains the user that created the connection, ABC (qse_student).
