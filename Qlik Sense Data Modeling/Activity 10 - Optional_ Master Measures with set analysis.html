Data Modeling for Qlik Sense

Activity 10 - Optional: Master Measures with set analysis

Summary:

  • Create two Master Measures within analysis mode, using set analysis syntax.
  • Leverage the two Master Measures inside KPI objects.

Module: Applying finishing touches

Qlik Sense app: ABC Sales Data

Challenge

In the ABC Sales Data app, create two new Master Measures, based on more complex expressions, by doing the following:

  • From the Assets panel, create a new Master Measure and use a set analysis Expression to only calculate America's number of orders.
  • Use Orders America as the Master Measure Name.
  • Repeat the process to create an Orders EMEA Master Measure. Use set analysis for that as well.
  • Create two KPI objects on the sheet, for Orders America and Orders EMEA respectively, to show the results.

Hints

  • The America expression calculates the number of orders for both North and South America divisions.
  • Orders America:
    Count({$<DivisionName={"*America*"}>} OrderRecordCounter)
  • Orders EMEA:
    Count({$<DivisionName={'EMEA'}>} OrderRecordCounter)

 

Step-by-step guide
  1. Edit the Master Items sheet. Create a new Master Measure using the following Expression:
    Count({$<DivisionName={"*America*"}>} OrderRecordCounter)
  2. Use Orders America as the Master Measure Name, and set the Description to Orders in America region.
  3. Add two Tags: Orders and America. Then, click the Create button.
  4. Create a second Master Measure using the following Expression:
    Count({$<DivisionName={'EMEA'}>} OrderRecordCounter)
  5. Name the Master Measure Orders EMEA and set the Description to Orders in EMEA region.
  6. Add two Tags: Orders and EMEA. Then, click the Create button.
  7. From the Assets panel, add two KPI objects onto the sheet.
  8. Add the Orders America Master Measure to the first KPI object and the Orders EMEA to the second KPI object.
  9. Disable the edit mode and observe the results.
