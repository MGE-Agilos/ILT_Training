Data Modeling for Qlik Sense

Activity 6: Master Dimensions from fields

Summary:

  • Create Master Dimension items for the application, via the Data model viewer.
  • Observe the items from the edit sheet view perspective.

Module: Applying finishing touches

Qlik Sense app: ABC Sales Data

Challenge

In the ABC Sales Data app, create three Master Dimensions by doing the following:

  • In your data model, preview the MasterCalendar table and select the Month field.
  • Use the Preview panel to add the Month field as a Master Dimension.
  • Add tags and a brief description for Month before you create the Master item.
  • Create two more Master Dimensions, Category from the CategoryName field and Product from ProductName.
  • In the App overview, open the first sheet. In edit mode, use the Assets panel to locate and verify that the Master Dimensions have been added.

Hints

  • The Master items can be reused multiple times within an app.
  • Adding tags to the Master items makes them more searchable.

 

Step-by-step guide
  1. In your data model, preview the MasterCalendar table and select the Month field.
  2. Use the Add as dimension button in the Preview panel to create a Master Dimension.
  3. Ensure the Name of the dimension is set to Month and enter Month of the OrderDate as Description.
  4. In the Tags input box, add the following tags: Calendar, Date, Month, and Year.
  5. Create the dimension. Look for the message to verify that the dimension has been created.
  6. While the Create new dimension window is still open, add a second Master Dimension for the CategoryName field. Set the Name to Category and the Description to Product category.
  7. Add the following two Tags: Product and Category. Then, click the Create button.
  8. Add a third Master Dimension for the ProductName field. Set the Name to Product and the Description to Product items from catalogue.
  9. Add the following three Tags: Category, Item, and Supplier. Then, click the Create button.
  10. Close the Create new dimension window and navigate to Analyze mode to open the first sheet.
  11. Enable the sheet edit mode and locate the Master items tab in the Assets panel.
  12. Open the Dimensions section to display the list of Master Dimensions. Then, click Month. A pop-up is displayed showing the attributes for this item.
  13. Once you finish observing the Master Items library, stop editing the sheet.

Solution

Click on the Solution image to open as a pop-up, or right-click on the image for more options. To close, click on the image again.
